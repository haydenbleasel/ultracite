Ultracite is ontworpen om direct te werken met monorepos. Hier leest u hoe u het met Turborepo gebruikt.

## Configuratie

Wanneer u Ultracite initialiseert, maakt het een `biome.jsonc`-bestand aan in de root van uw repository. Dit bestand wordt gebruikt om Ultracite te configureren voor uw gehele monorepo.

```
my-turborepo/
├── apps
├── packages
├── turbo.json
└── biome.jsonc
```

## Scripts toevoegen

Omdat Biome zo snel is, kunt u een root-script gebruiken in plaats van in elk pakket afzonderlijke scripts te maken.

U kunt de `check`- en `fix`-scripts toevoegen aan uw `package.json`-bestand.

```json title="package.json"
{
  "scripts": {
    "check": "ultracite check",
    "fix": "ultracite fix"
  }
}
```

Optioneel kunt u ook de `check`- en `fix`-scripts toevoegen aan uw `turbo.json`-bestand.

```json title="turbo.json"
{
  "tasks": {
    "//#check": {},
    "//#fix": {
      "cache": false
    }
  }
}
```

U kunt deze scripts nu uitvoeren met `turbo run check` en `turbo run fix`.
