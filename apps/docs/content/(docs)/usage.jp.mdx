## IDE 統合

Ultracite はエディタに統合して使用するのが最も効果的で、コードを書いている間に即時のフィードバックが得られます。保存時に自動で動作するよう設計されています。動作を試すには、IDE でプロジェクトを開いてください。ファイルを編集すると、保存時にフォーマットが適用されるはずです。小さなコードスタイルの間違い（余分なセミコロンやインデントの誤りなど）を入れて保存してみてください — Ultracite（Biome）が即座にファイルを再フォーマットします。

未使用の変数のような一般的なリント問題を導入すると、波線の下線や問題パネルでの警告が表示されることがあります。保存時に、その問題が自動修正可能であれば Ultracite は修正を試みます（例えば未使用の import を削除するなど）。それ以外の場合は、対処が必要な警告を残します。

### 即時フォーマット

ファイルを保存するたびに、Biome（Ultracite の設定で）がファイルをフォーマットします。別途 Prettier を実行したりフォーマットを気にする必要はありません — 自動的に処理されます。

`codeActionsOnSave` 設定のおかげで、VS Code は保存時に Biome の「fix all」アクションも適用します。これは、自動修正を持つ任意のリントルール（`!=` を `!==` に変換する、欠落している括弧を追加する、import の順序を修正するなど）が適用されることを意味します。まるで ESLint で `--fix` を自動的に実行したかのようです。

`formatOnPaste` も有効にしているため、他所からコードを貼り付けた場合でも、すぐにスタイルに合わせてフォーマットされます。

ここでの主な利点は即時フィードバックです。VS Code でコードを書くと、Ultracite が継続的にコードをクリーンに保ちます。時間が経つにつれて、リントエラーの修正に費やす時間はほとんどかからなくなるでしょう — Ultracite はそれらを自動で修正するか、早期に指摘してくれます。これにより、コードスタイルや軽微な問題が蓄積しない、スムーズな開発ワークフローが実現します。

### 問題パネルとの統合

注意を要する残りの問題はすべて問題パネルに表示されます。例えば、Ultracite によって禁止されている何かが見つかった場合や、修正できないエラーが見つかった場合（非推奨の API を使用した、Promise の拒否を処理し忘れた、など）は、それがエラーや警告として一覧表示されます。クリックするとコード内の該当箇所にジャンプできます。

### コードレンズ / クイックフィックス

多くの場合、Biome の VS Code 拡張はクイックフィックスの提案を提供します。黄色い電球アイコンや提案ポップアップが表示されたら、手動で修正を適用することもできます。たとえば、未使用の import がある場合、（保存時に既に削除されていなければ）それを削除するクイックフィックスが提示されることがあります。

## CLI の使用方法

Ultracite には便利な CLI（インストール時に `ultracite` として呼び出します）が付属しています。ここでは一般的な使用方法を示します。

### コードのチェック

`check` コマンドは、ファイルを修正せずにリンタを実行する Biome の `check` コマンドをラップします。

```bash title="Terminal"
npx ultracite check
```

`--diagnostic-level` フィルタを渡して、エラーまたは警告のみを表示することができます:

```bash title="Terminal"
# Show only errors
npx ultracite check --diagnostic-level error

# Show warnings and above (errors)
npx ultracite check --diagnostic-level warn
```

### コードの修正

`fix` コマンドはファイルを修正して実行する Biome の `check --write` コマンドをラップします。

```bash title="Terminal"
npx ultracite fix
```

また、次のようにして安全でない修正を適用することもできます:

```bash title="Terminal"
npx ultracite fix --unsafe
```

### セットアップの検証

`doctor` コマンドはセットアップに問題がないかをチェックし、推奨事項を提供します。Ultracite をインストールした後に、すべてが正しく設定されているかを確認するために実行するのに便利です。

```bash title="Terminal"
npx ultracite doctor
```
