Il **Protocollo Model Context (MCP)** è uno standard aperto che permette agli assistenti AI come Claude, Cursor e altri strumenti basati sull'AI di connettersi in modo sicuro con sorgenti di dati esterne e sistemi. Pensalo come un "telecomando universale" che consente ai tuoi strumenti AI di accedere a dati e funzionalità del mondo reale.

Ultracite supporta MCP per potenziare il tuo flusso di lavoro di sviluppo AI.

## Guida all'installazione

### Passo 1: Scegli il tuo strumento AI

Per prima cosa, assicurati di utilizzare uno strumento di sviluppo AI che supporti MCP:

- [Claude Desktop](https://claude.ai/download) (Gratuito - consigliato per i principianti)
- [Cursor](https://www.cursor.com/) (Editor di codice potenziato dall'AI)
- [Windsurf by Codeium](https://windsurf.com/) (Piattaforma di sviluppo AI)
- Altri strumenti compatibili con MCP

### Passo 2: Individua il file di configurazione

A seconda del tuo strumento AI, dovrai modificare uno di questi file:

- **Claude Desktop**: `~/Library/Application\ Support/Claude/claude_desktop_config.json` per macOS e `%APPDATA%\Claude\claude_desktop_config.json` per Windows
- **Cursor**: `.cursor/mcp.json`
- **Windsurf**: `.codeium/windsurf/mcp_config.json`
- **Altri strumenti**: Controlla la documentazione MCP del tuo strumento

### Passo 3: Aggiungi la configurazione di Ultracite

Copia e incolla questa configurazione nel tuo file di configurazione MCP:

```json
{
  "mcpServers": {
    "ultracite": {
      "command": "npx",
      "args": [
        "-y",
        "mcp-remote",
        "https://www.ultracite.ai/api/mcp/mcp"
      ]
    }
  }
}
```

### Passo 4: Riavvia il tuo strumento AI

Chiudi e riapri l'applicazione AI affinché le modifiche abbiano effetto.

### Passo 5: Verifica la connessione

Testa l'integrazione chiedendo al tuo assistente AI:

> Quali regole di Ultracite sono disponibili?

Se la procedura ha successo, la tua AI dovrebbe riuscire a elencare e spiegare le regole di Ultracite!

## Più server MCP

Puoi usare Ultracite insieme ad altri server MCP:

```json
{
  "mcpServers": {
    "ultracite": {
      "command": "npx",
      "args": ["-y", "mcp-remote", "https://www.ultracite.ai/api/mcp/mcp"]
    },
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"]
    },
    "filesystem": {
      "command": "npx", 
      "args": ["-y", "@modelcontextprotocol/server-filesystem", "/path/to/project"]
    }
  }
}
```

## Esempi d'uso

### Esplorare le regole disponibili

Chiedi al tuo assistente AI:

> Quali regole ha Ultracite per i React hooks?

La risposta prevista includerà le regole di linting relative ai React hook e le loro descrizioni pertinenti.

### Comprendere le categorie di regole

> Mostrami tutte le regole di accessibilità in Ultracite

La tua AI può elencare e spiegare le regole di accessibilità (a11y) configurate in Ultracite.

### Confronto tra preset

> Qual è la differenza tra ultracite/core e ultracite/react?

Ottieni informazioni dettagliate sulle diverse configurazioni preset e su quando utilizzarle.

## Sicurezza e Privacy

### Gestione dei dati

- Il server MCP di Ultracite fornisce solo documentazione pubblica sulle regole di linting e informazioni di configurazione
- Nessun dato personale o codice viene trasmesso ai nostri server
- Tutta la comunicazione avviene attraverso il livello di sicurezza dello strumento AI che hai scelto

### Autenticazione

- Non è richiesta autenticazione per l'accesso alla documentazione pubblica
- Funzionalità premium future potrebbero richiedere chiavi API
- Usa sempre gli endpoint MCP ufficiali di Ultracite
