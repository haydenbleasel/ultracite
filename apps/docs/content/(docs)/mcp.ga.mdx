Is é an **Prótacal Comhthéacs Múnla (MCP)** caighdeán oscailte a cheadaíonn cúnamhaithe AI cosúil le Claude, Cursor, agus uirlisí eile faoi thiomáint AI nasc a dhéanamh go slán le foinsí sonraí agus córais seachtracha. Smaoinigh air mar "riomhaire iargúlta ollmhór" a ligeann do d’uirlisí AI rochtain a fháil ar shonraí agus feidhmiúlacht an domhain fhíor.

Tacaíonn Ultracite le MCP chun do sreabhadh oibre forbartha AI a chumhachtú.

## Treoir Suiteála

### Céim 1: Roghnaigh Do Uirlis AI

Ar an gcéad dul síos, déan cinnte go bhfuil tú ag úsáid uirlis forbartha AI a thacaíonn le MCP:

- [Claude Desktop](https://claude.ai/download) (Saor in aisce - molta do thosaitheoirí)
- [Cursor](https://www.cursor.com/) (eagarthóir cód faoi thiomáint AI)
- [Windsurf by Codeium](https://windsurf.com/) (ardán forbartha AI)
- Uirlisí eile atá comhoiriúnach le MCP

### Céim 2: Aimsiú Do Cháipín Cumraíochta

Ag brath ar do uirlis AI, beidh ort ceann de na comhaid seo a eagarthóireacht:

- **Claude Desktop**: `~/Library/Application\ Support/Claude/claude_desktop_config.json` do macOS agus `%APPDATA%\Claude\claude_desktop_config.json` do Windows
- **Cursor**: `.cursor/mcp.json`
- **Windsurf**: `.codeium/windsurf/mcp_config.json`
- **Uirlisí eile**: Seiceáil do dhoiciméadú MCP don uirlis

### Céim 3: Cuir Cumraíocht Ultracite leis

Cóipeáil agus greamaigh an cumraíocht seo i do chomhad cumraíochta MCP:

```json
{
  "mcpServers": {
    "ultracite": {
      "command": "npx",
      "args": [
        "-y",
        "mcp-remote",
        "https://www.ultracite.ai/api/mcp/mcp"
      ]
    }
  }
}
```

### Céim 4: Atosaigh Do Uirlis AI

Dún agus athoscail do fheidhmchlár AI ionas go dtiocfaidh na hathruithe i ngníomh.

### Céim 5: Fíoraigh an Nasc

Tástáil an comhtháthaithe trí iarraidh ar do chúnamhóir AI:

> Cé na rialacha Ultracite atá ar fáil?

Má éiríonn leis, ba chóir go mbeifeadh d’AI ábalta rialacha Ultracite a liostáil agus a mhíniú!

## Il-Fhreastalaithe MCP

Is féidir leat Ultracite a úsáid in éineacht le freastalaithe MCP eile:

```json
{
  "mcpServers": {
    "ultracite": {
      "command": "npx",
      "args": ["-y", "mcp-remote", "https://www.ultracite.ai/api/mcp/mcp"]
    },
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"]
    },
    "filesystem": {
      "command": "npx", 
      "args": ["-y", "@modelcontextprotocol/server-filesystem", "/path/to/project"]
    }
  }
}
```

## Samplaí Úsáide

### Ag Iniúchadh na Rialacha atá Ar Fáil

Iarr ar d’cúnamhóir AI:

> Cén rialacha atá ag Ultracite maidir le React hooks?

Ba chóir don fhreagra a mbeifí ag súil leis a áireamh na rialacha linting ábhartha do React hooks agus a míniúcháin.

### Tuiscint ar Chatagóirí Rialacha

> Taispeáin dom na rialacha uile maidir le inrochtaineacht in Ultracite

Is féidir le do AI na rialacha a11y atá cumraithe in Ultracite a liostáil agus a mhíniú.

### Comparáid Réamhshocruithe

> Cad é an difríocht idir ultracite/core agus ultracite/react?

Faigh faisnéis mhionsonraithe faoi chumraíochtaí réamhshocraithe éagsúla agus cathain is fearr iad a úsáid.

## Slándáil agus Príobháideacht

### Láimhseáil Sonraí

- Ní sholáthraíonn freastalaí MCP Ultracite ach doiciméadú poiblí rialacha linting agus eolas cumraíochta
- Níl aon shonraí pearsanta ná cód á n-aistriú chuig ár bhfreastalaithe
- Tarlaíonn gach cumarsáid tríd an sraith slándála den uirlis AI a roghnaigh tú

### Fíordheimhniú

- Níl gá le fíordheimhniú chun rochtain a fháil ar an ndoiciméadú poiblí
- D'fhéadfadh go mbeadh eochracha API ag teastáil do ghnéithe préimhe amach anseo
- Bain úsáid i gcónaí as na endpoints MCP oifigiúla Ultracite
