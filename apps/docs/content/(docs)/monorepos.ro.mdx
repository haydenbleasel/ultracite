---
title: Monorepo-uri
description: Cum să folosești Ultracite într-un monorepo.
---

Ultracite este proiectat să funcționeze imediat cu monorepo-uri. Iată cum să-l folosești cu Turborepo.

## Configurare

Când inițializezi Ultracite, acesta va crea un fișier `biome.jsonc` în rădăcina depozitului tău. Acest fișier va fi folosit pentru a configura Ultracite pentru întregul tău monorepo.

```
my-turborepo/
├── apps
├── packages
├── turbo.json
└── biome.jsonc
```

## Adăugarea scripturilor

Deoarece Biome este atât de rapid, poți folosi un script la nivelul rădăcinii în loc să creezi scripturi separate în fiecare dintre pachetele tale.

Poți adăuga scripturile `check` și `fix` în fișierul tău `package.json`.

```json title="package.json"
{
  "scripts": {
    "check": "ultracite check",
    "fix": "ultracite fix"
  }
}
```

Opțional, poți adăuga de asemenea scripturile `check` și `fix` în fișierul tău `turbo.json`.

```json title="turbo.json"
{
  "tasks": {
    "//#check": {},
    "//#fix": {
      "cache": false
    }
  }
}
```

Acum vei putea rula aceste scripturi folosind `turbo run check` și `turbo run fix`.
