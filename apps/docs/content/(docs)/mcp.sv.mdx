Det **Modellkontextprotokollet (MCP)** är en öppen standard som låter AI-assistenter som Claude, Cursor och andra AI-drivna verktyg ansluta säkert till externa datakällor och system. Tänk på det som en "universell fjärrkontroll" som låter dina AI-verktyg få tillgång till verkliga data och funktionalitet.

Ultracite stöder MCP för att påskynda din AI-utvecklingsarbetsflöde.

## Installationsguide

### Steg 1: Välj ditt AI-verktyg

Först, se till att du använder ett AI-utvecklingsverktyg som stödjer MCP:

- [Claude Desktop](https://claude.ai/download) (Gratis - rekommenderas för nybörjare)
- [Cursor](https://www.cursor.com/) (AI-drivet kodredigeringsverktyg)
- [Windsurf by Codeium](https://windsurf.com/) (AI-utvecklingsplattform)
- Andra verktyg som är kompatibla med MCP

### Steg 2: Lokalisera din konfigurationsfil

Beroende på ditt AI-verktyg behöver du redigera en av dessa filer:

- **Claude Desktop**: `~/Library/Application\ Support/Claude/claude_desktop_config.json` för macOS och `%APPDATA%\Claude\claude_desktop_config.json` för Windows
- **Cursor**: `.cursor/mcp.json`
- **Windsurf**: `.codeium/windsurf/mcp_config.json`
- **Andra verktyg**: Kontrollera ditt verktygs MCP-dokumentation

### Steg 3: Lägg till Ultracite-konfiguration

Kopiera och klistra in denna konfiguration i din MCP-konfigurationsfil:

```json
{
  "mcpServers": {
    "ultracite": {
      "command": "npx",
      "args": [
        "-y",
        "mcp-remote",
        "https://www.ultracite.ai/api/mcp/mcp"
      ]
    }
  }
}
```

### Steg 4: Starta om ditt AI-verktyg

Stäng och öppna din AI-applikation igen för att ändringarna ska börja gälla.

### Steg 5: Verifiera anslutningen

Testa integrationen genom att fråga din AI-assistent:

> Vilka Ultracite-regler är tillgängliga?

Om det lyckas bör din AI kunna lista och förklara Ultracite-regler!

## Flera MCP-servrar

Du kan använda Ultracite tillsammans med andra MCP-servrar:

```json
{
  "mcpServers": {
    "ultracite": {
      "command": "npx",
      "args": ["-y", "mcp-remote", "https://www.ultracite.ai/api/mcp/mcp"]
    },
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"]
    },
    "filesystem": {
      "command": "npx", 
      "args": ["-y", "@modelcontextprotocol/server-filesystem", "/path/to/project"]
    }
  }
}
```

## Användningsexempel

### Utforska tillgängliga regler

Fråga din AI-assistent:

> Vilka regler har Ultracite för React-hooks?

Förväntat svar kommer att inkludera relevanta lint-regler för React-hooks och deras beskrivningar.

### Förstå regelkategorier

> Visa alla tillgänglighetsregler i Ultracite

Din AI kan lista och förklara tillgänglighetsreglerna (a11y) som är konfigurerade i Ultracite.

### Jämföra förinställningar

> Vad är skillnaden mellan ultracite/core och ultracite/react?

Få detaljerad information om olika förinställningar och när de bör användas.

## Säkerhet och integritet

### Hantering av data

- Ultracite MCP-servern tillhandahåller endast offentlig dokumentation om lint-regler och konfigurationsinformation
- Ingen personlig data eller kod överförs till våra servrar
- All kommunikation sker genom ditt valda AI-verktygs säkerhetslager

### Autentisering

- Ingen autentisering krävs för åtkomst till offentlig dokumentation
- Framtida premiumfunktioner kan kräva API-nycklar
- Använd alltid officiella Ultracite MCP-slutpunkter
