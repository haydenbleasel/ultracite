---
title: Hooks
description: Hvordan du integrerer Ultracite med AI-kodeassistenter gennem editor-hooks.
---

Ultracite kan automatisk formatere og rette kode efter at AI-kodeassistenter har foretaget ændringer gennem editor-hooks. Dette sikrer, at al AI-genereret kode overholder dit projekts standarder uden manuel indgriben.

Hooks fungerer sammen med Biome's linting og formatering. Efter at en AI-kodeassistent har redigeret en fil, kører hooket automatisk `npx ultracite fix` for at formatere og rette lint-problemer, der kan auto-rettes. Denne dobbelte tilgang sikrer både intelligent kodegenerering (gennem AI-regler) og automatiseret kodekvalitet (gennem hooks).

> **Bemærk:** Hooks er adskilt fra AI-kodningsregler. Flaget `--agents` konfigurerer AI-kodningsreglerne, mens flaget `--hooks` opsætter automatisk formatering efter filændringer.

## Opsætning

### Automatisk opsætning

Når du kører `npx ultracite init`, bliver du spurgt, hvilke editor-hooks du ønsker at aktivere:

```bash
? Which agent hooks do you want to enable (optional)?
  ◯ Cursor
  ◯ Claude Code
```

Dette vil oprette den relevante hook-konfigurationsfil i din projektmappe.

### Manuel opsætning

Hvis du ønsker at sætte det op manuelt eller tilføje hooks efter initialisering, kør:

```bash
npx ultracite init --hooks cursor,claude
```

Alternativt kan du manuelt oprette eller redigere hook-konfigurationsfilerne i din projektmappe og tilføje den passende hook-kommando.

## Understøttede editorer

| Editor      | Hook-filplacering          | Dokumentation                                                           |
| ----------- | --------------------------- | ----------------------------------------------------------------------- |
| Cursor      | `.cursor/hooks.json`        | [Cursor-hooks](https://cursor.com/docs/agent/hooks)                    |
| Claude Code | `.claude/settings.json`     | [Claude Code-hooks](https://docs.claude.com/en/docs/claude-code/hooks) |

## Sådan fungerer det

1. **Automatisk udførelse**: Efter AI-agenten redigerer en fil, kører hooket automatisk `npx ultracite fix`  
2. **Kodeformatering**: Ultracite formaterer den redigerede kode i overensstemmelse med din Biome-konfiguration  
3. **Lintrettelser**: Lint-fejl, der kan rettes automatisk, bliver rettet automatisk  
4. **Sømløs integration**: Processen foregår transparent uden at afbryde din arbejdsgang

## Fordele

Ved at bruge Ultracite editor-hooks får du flere fordele:

- **Konsistens**: Al AI-genereret kode bliver automatisk formateret, så den matcher din stil  
- **Kvalitet**: Lint-fejl rettes umiddelbart efter kodegenereringen  
- **Effektivitet**: Ingen grund til manuelt at køre formateringsværktøjer eller linters efter AI-ændringer  
- **Standarder**: Sikrer, at AI-genereret kode altid overholder projektets standarder  
- **Rene commits**: Koden er korrekt formateret, inden du overhovedet gennemgår den

## Tilpasning

Ultracite flettes intelligent sammen med eksisterende hook-konfigurationer:

- **Bevarer eksisterende hooks**: Alle eksisterende hooks og deres kommandoer forbliver intakte  
- **Undgår duplikater**: Hvis Ultracite allerede er konfigureret, bliver det ikke tilføjet igen  
- **Ikke-destruktiv**: Din eksisterende hook-konfigurationsstruktur bevares

Du kan også tilpasse hook-adfærden ved at redigere hook-konfigurationsfilerne direkte, hvis du har brug for andre formateringskommandoer eller yderligere handlinger efter redigering.
