Ultracite kan automatisch code formatteren en repareren nadat AI-ontwikkelhulpjes bewerkingen hebben uitgevoerd via editor-hooks. Dit zorgt ervoor dat alle door AI gegenereerde code voldoet aan de standaarden van je project zonder handmatige tussenkomst.

De hooks werken naast Biome's linting en formatting. Nadat een AI-ontwikkelhulp een bestand bewerkt, voert de hook automatisch `npx ultracite fix` uit om automatisch oplosbare lintingproblemen te formatteren en te verhelpen. Deze dubbele aanpak zorgt zowel voor intelligente codegeneratie (via AI-regels) als geautomatiseerde codekwaliteit (via hooks).

> **Opmerking:** Hooks zijn los van AI-coderegels. De `--agents` vlag configureert AI-coderegels, terwijl de `--hooks` vlag automatische formattering na bestandbewerkingen instelt.

## Installatie

### Automatische installatie

Wanneer je `npx ultracite init` uitvoert, wordt je gevraagd welke editor-hooks je wilt inschakelen:

```bash
? Which agent hooks do you want to enable (optional)?
  ◯ Cursor
  ◯ Claude Code
```

Dit maakt het betreffende configuratiebestand voor hooks in je projectmap aan.

### Handmatige installatie

Als je handmatig wilt instellen of hooks wilt toevoegen nadat je hebt geïnitialiseerd, voer dan uit:

```bash
npx ultracite init --hooks cursor,claude
```

Als alternatief kun je handmatig de hookconfiguratiebestanden in je projectmap aanmaken of bewerken en het juiste hook-commando toevoegen.

## Hoe het werkt

1. **Automatische uitvoering**: Nadat de AI-agent een bestand heeft bewerkt, voert de hook automatisch `npx ultracite fix` uit  
2. **Codeformatteren**: Ultracite formatteert de bewerkte code volgens je Biome-configuratie  
3. **Linting-fixes**: Automatisch te herstellen lintingproblemen worden automatisch opgelost  
4. **Naadloze integratie**: Het proces verloopt transparant zonder je workflow te onderbreken

## Voordelen

Het gebruik van Ultracite-editorhooks biedt verschillende voordelen:

- **Consistentie**: Alle door AI gegenereerde code wordt automatisch geformatteerd zodat deze overeenkomt met je stijl  
- **Kwaliteit**: Lintingproblemen worden direct na codegeneratie verholpen  
- **Efficiëntie**: Geen noodzaak om handmatig formatters of linters uit te voeren na AI-bewerkingen  
- **Standaarden**: Zorgt ervoor dat door AI gegenereerde code altijd voldoet aan projectstandaarden  
- **Schone commits**: Code is correct geformatteerd nog voordat je deze beoordeelt

## Aanpassing

Ultracite voegt zich intelligent naar bestaande hookconfiguraties:

- **Behoudt bestaande hooks**: Alle bestaande hooks en hun commando's blijven intact  
- **Voorkomt duplicaten**: Als Ultracite al is geconfigureerd, wordt het niet opnieuw toegevoegd  
- **Niet-destructief**: Je bestaande hookconfiguratiestructuur blijft behouden

Je kunt het hookgedrag ook aanpassen door de hookconfiguratiebestanden direct te wijzigen als je andere formatteringscommando's of aanvullende acties na bewerking nodig hebt.
