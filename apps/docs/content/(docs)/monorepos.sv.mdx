---

Ultracite är utformat för att fungera direkt med monorepos. Så här använder du det med Turborepo.

## Konfiguration

När du initierar Ultracite skapas en `biome.jsonc`-fil i roten av ditt repository. Denna fil kommer att användas för att konfigurera Ultracite för hela ditt monorepo.

```
my-turborepo/
├── apps
├── packages
├── turbo.json
└── biome.jsonc
```

## Lägga till skript

Eftersom Biome är så snabbt kan du använda ett skript i rotkatalogen istället för att skapa separata skript i varje paket.

Du kan lägga till skripten `check` och `fix` i din `package.json`-fil.

```json title="package.json"
{
  "scripts": {
    "check": "ultracite check",
    "fix": "ultracite fix"
  }
}
```

Valfritt kan du även lägga till skripten `check` och `fix` i din `turbo.json`-fil.

```json title="turbo.json"
{
  "tasks": {
    "//#check": {},
    "//#fix": {
      "cache": false
    }
  }
}
```

Du kan nu köra dessa skript med `turbo run check` och `turbo run fix`.
