---
title: Asennus
description: Kuinka asentaa Ultracite projektiisi.
---

Ultracite voidaan asentaa kaikilla suosituilla pakettienhallinoilla. Tarvitset Node.js v14.18+ (suositellaan Node 18+) ja projektin, jossa on `package.json`-tiedosto.

## Asennus

Helpoin tapa aloittaa on suorittaa Ultraciten `init`-skripti, joka asentaa tarvittavat paketit ja luo konfiguraatiotiedoston puolestasi. Projektihakemistossasi suorita:

```package-install
npx ultracite init
```

Voit halutessasi välittää seuraavat liput `init`-komennolle:

| Lipuke              | Kuvaus                                           | Vaihtoehdot                                                              |
|---------------------|--------------------------------------------------|-------------------------------------------------------------------------|
| `--pm`              | Käytettävä pakettienhallinta.                     | `pnpm`, `bun`, `yarn`, `npm`                                            |
| `--frameworks`      | Konfiguroitavat kehykset.                         | Pilkuilla eroteltuna: `react`, `next`, `solid`, `vue`, `qwik`, `angular`, `remix`, `svelte` |
| `--editors`         | Konfiguroitavat editorit.                         | Pilkuilla eroteltuna: `vscode`, `zed`                                    |
| `--agents`          | Konfiguroitavat tekoälyagentit.                   | Pilkuilla eroteltuna: `vscode-copilot`, `cursor`, `windsurf`, `zed`, `claude`, `codex`, `kiro`, `cline`, `amp`, `aider`, `firebase-studio`, `open-hands`, `gemini-cli`, `junie`, `augmentcode`, `kilo-code`, `goose`, `roo-code`, `warp`, `droid` |
| `--hooks`           | Konfiguroitavat agenttien hookit.                 | Pilkuilla eroteltuna: `cursor`, `claude`                                 |
| `--integrations`    | Asennettavat lisäintegraatiot.                    | Pilkuilla eroteltuna: `husky`, `lefthook`, `lint-staged`                 |
| `--migrate`         | Poistettavat migraatiotyökalut.                    | Pilkuilla eroteltuna: `eslint`, `prettier` (poistaa riippuvuudet, konfiguraatiotiedostot ja editorin asetukset) |
| `--skip-install`    | Ohita riippuvuuksien asentaminen.                  | (arvoa ei tarvita)                                                       |
| `--quiet`           | Estä interaktiiviset kehotteet ja visuaalinen tulostus. | (arvoa ei tarvita, automaattisesti käytössä CI:ssä)                     |

### Ohjelmallinen käyttö

Työkaluille, jotka tarvitsevat kutsua Ultracitea ohjelmallisesti (esim. scaffolding-CLI:t), voit käyttää `--quiet`-lippua estämään kaikki interaktiiviset kehotteet ja visuaalisen tulostuksen:

```bash title="Terminal"
npx ultracite init --quiet --pm npm
```

`--quiet`-lippu otetaan automaattisesti käyttöön, kun `CI`-ympäristömuuttuja on asetettu arvoon `true` tai `1`, mikä tekee siitä sopivan CI/CD-putkistoihin:

```bash title="Terminal"
CI=true npx ultracite init --pm npm
```

Hiljaisessa tilassa Ultracite:
- Ohittaa kaikki interaktiiviset kehotteet
- Käyttää järkeviä oletusasetuksia kaikille vaihtoehdoille
- Piilottaa ASCII-artin, spinnersit ja lokiviestit
- Luo vain ydin Biome-konfiguraation
- Poistuu siististi koodilla 0 onnistuneessa tilanteessa tai 1 epäonnistumisessa

## Laajennukset

Jos käytät Visual Studio Codea, asenna virallinen [Biome](https://marketplace.visualstudio.com/items?itemName=biomejs.biome) -laajennus, joka mahdollistaa tallennettaessa suoritettavan muotoilun/korjauksen:

```bash title="Terminal"
code --install-extension biomejs.biome
```

Lisäksi, jos käytät Tailwind CSS:ää, suositellaan asentamaan [Tailwind CSS IntelliSense](https://marketplace.visualstudio.com/items?itemName=bradlc.vscode-tailwindcss) -laajennus luokkien lajittelua ja vihjeitä varten:

```bash title="Terminal"
code --install-extension bradlc.vscode-tailwindcss
```
