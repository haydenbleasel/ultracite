O **Model Context Protocol (MCP)** é um standard aberto que permite que assistentes de IA como Claude, Cursor e outras ferramentas alimentadas por IA se conectem de forma segura a fontes de dados e sistemas externos. Pense nele como um "controlo remoto universal" que permite às suas ferramentas de IA aceder a dados e funcionalidades do mundo real.

O Ultracite suporta MCP para potenciar o seu fluxo de trabalho de desenvolvimento de IA.

## Guia de Instalação

### Passo 1: Escolha a sua ferramenta de IA

Primeiro, certifique-se de que está a utilizar uma ferramenta de desenvolvimento de IA que suporte MCP:

- [Claude Desktop](https://claude.ai/download) (Gratuito - recomendado para principiantes)
- [Cursor](https://www.cursor.com/) (editor de código com IA)
- [Windsurf by Codeium](https://windsurf.com/) (plataforma de desenvolvimento com IA)
- Outras ferramentas compatíveis com MCP

### Passo 2: Localize o seu ficheiro de configuração

Dependendo da sua ferramenta de IA, terá de editar um destes ficheiros:

- **Claude Desktop**: `~/Library/Application\ Support/Claude/claude_desktop_config.json` para macOS e `%APPDATA%\Claude\claude_desktop_config.json` para Windows
- **Cursor**: `.cursor/mcp.json`
- **Windsurf**: `.codeium/windsurf/mcp_config.json`
- **Outras ferramentas**: Consulte a documentação MCP da sua ferramenta

### Passo 3: Adicionar a configuração do Ultracite

Copie e cole esta configuração no seu ficheiro de configuração MCP:

```json
{
  "mcpServers": {
    "ultracite": {
      "command": "npx",
      "args": [
        "-y",
        "mcp-remote",
        "https://www.ultracite.ai/api/mcp/mcp"
      ]
    }
  }
}
```

### Passo 4: Reinicie a sua ferramenta de IA

Feche e abra novamente a sua aplicação de IA para que as alterações tenham efeito.

### Passo 5: Verifique a ligação

Teste a integração perguntando ao seu assistente de IA:

> What Ultracite rules are available?

Se for bem-sucedido, o seu assistente de IA deverá conseguir listar e explicar as regras do Ultracite!

## Vários servidores MCP

Pode usar o Ultracite juntamente com outros servidores MCP:

```json
{
  "mcpServers": {
    "ultracite": {
      "command": "npx",
      "args": ["-y", "mcp-remote", "https://www.ultracite.ai/api/mcp/mcp"]
    },
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"]
    },
    "filesystem": {
      "command": "npx", 
      "args": ["-y", "@modelcontextprotocol/server-filesystem", "/path/to/project"]
    }
  }
}
```

## Exemplos de Uso

### Explorar regras disponíveis

Pergunte ao seu assistente de IA:

> What rules does Ultracite have for React hooks?

A resposta esperada incluirá as regras de lint relevantes para React hooks e as suas descrições.

### Compreender categorias de regras

> Show me all accessibility rules in Ultracite

O seu assistente de IA pode listar e explicar as regras de a11y configuradas no Ultracite.

### Comparar presets

> What's the difference between ultracite/core and ultracite/react?

Obtenha informações detalhadas sobre diferentes configurações de presets e quando as deve utilizar.

## Segurança e Privacidade

### Tratamento de Dados

- O servidor MCP do Ultracite fornece apenas documentação pública das regras de lint e informação de configuração
- Não são transmitidos dados pessoais nem código para os nossos servidores
- Toda a comunicação ocorre através da camada de segurança da sua ferramenta de IA escolhida

### Autenticação

- Não é necessária autenticação para aceder à documentação pública
- Funcionalidades premium futuras podem requerer chaves de API
- Use sempre endpoints MCP oficiais do Ultracite
