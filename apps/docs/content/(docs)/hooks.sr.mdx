title: Hookovi
description: Kako integrisati Ultracite sa AI asistentima za kodiranje putem hookova u editoru.
---

Ultracite može automatski formatirati i popraviti kod nakon što AI asistenti za kodiranje izvrše izmene putem hookova u editoru. Ovo osigurava da sav AI-generisani kod bude u skladu sa standardima vašeg projekta bez ručne intervencije.

Hookovi rade zajedno sa Biome-ovim lintovanjem i formatiranjem. Nakon što AI asistent izmeni fajl, hook automatski pokreće `npx ultracite fix` da bi formatirao i popravio lint greške koje se mogu automatski ispraviti. Ovaj dvostruki pristup osigurava i inteligentno generisanje koda (preko AI pravila) i automatizovan kvalitet koda (preko hookova).

> **Napomena:** Hookovi su odvojeni od pravila AI kodiranja. Zastavica `--agents` konfiguriše pravila AI kodiranja, dok zastavica `--hooks` podešava automatsko formatiranje nakon izmena fajla.

## Podešavanje

### Automatsko podešavanje

When you run `npx ultracite init`, you'll be prompted to select which editor hooks you want to enable:

```bash
? Which agent hooks do you want to enable (optional)?
  ◯ Cursor
  ◯ Claude Code
```

Ovo će kreirati odgovarajući konfiguracioni fajl za hook u vašem projektnom direktorijumu.

### Ručno podešavanje

Ako želite ručno da postavite stvari ili dodate hookove nakon inicijalizacije, pokrenite:

```bash
npx ultracite init --hooks cursor,claude
```

Alternativno, možete ručno kreirati ili izmeniti konfiguracione fajlove hookova u vašem projektnom direktorijumu i dodati odgovarajuću komandu hooka.

## Podržani editori

| Editor      | Hook File Location          | Dokumentacija                                                           |
| ----------- | --------------------------- | ----------------------------------------------------------------------- |
| Cursor      | `.cursor/hooks.json`        | [Hookovi za Cursor](https://cursor.com/docs/agent/hooks)                |
| Claude Code | `.claude/settings.json`     | [Hookovi za Claude Code](https://docs.claude.com/en/docs/claude-code/hooks) |

## Kako radi

1. **Automatsko izvršavanje**: Nakon što AI agent izmeni fajl, hook automatski pokreće `npx ultracite fix`  
2. **Formatiranje koda**: Ultracite formatira izmenjeni kod u skladu sa vašom Biome konfiguracijom  
3. **Ispravke lintinga**: Auto-popravljive lint greške se automatski rešavaju  
4. **Besprekorna integracija**: Proces se odvija transparentno bez prekidanja vašeg radnog toka

## Prednosti

Korišćenje Ultracite hookova u editoru pruža nekoliko prednosti:

- **Doslednost**: Sav AI-generisani kod se automatski formatira da odgovara vašem stilu  
- **Kvalitet**: Greške u lintovanju se popravljaju odmah nakon generisanja koda  
- **Efikasnost**: Nema potrebe ručno pokretati formatere ili lintere nakon AI izmena  
- **Standardi**: Osigurava da AI-generisani kod uvek poštuje projektne standarde  
- **Čisti commitovi**: Kod je pravilno formatiran pre nego što ga pregledate

## Prilagođavanje

Ultracite se inteligentno uklapa sa postojećim konfiguracijama hookova:

- **Zadržava postojeće hookove**: Svi postojeći hookovi i njihove komande ostaju netaknuti  
- **Izbegava duplikate**: Ako je Ultracite već konfigurisano, neće biti dodato ponovo  
- **Nedestruktivno**: Struktura vaše postojeće konfiguracije hookova se održava

Takođe možete prilagoditi ponašanje hookova tako što ćete direktno izmeniti konfiguracione fajlove hookova ako su vam potrebne drugačije komande za formatiranje ili dodatne akcije nakon izmene.
