The **Protokol modelnega konteksta (MCP)** je odprt standard, ki AI asistentom, kot so Claude, Cursor in drugim orodjem, podprtih z AI, omogoča varno povezavo z zunanjimi podatkovnimi viri in sistemi. Razmišljajte o njem kot o "univerzalnem daljinskem upravljalniku", ki vašim AI orodjem omogoči dostop do podatkov in funkcionalnosti iz resničnega sveta.

Ultracite podpira MCP za pospešitev vašega delovnega toka razvoja AI.

## Vodnik za namestitev

### Korak 1: Izberite AI orodje

Najprej se prepričajte, da uporabljate orodje za razvoj AI, ki podpira MCP:

- **Claude Desktop** (Brezplačno - priporočeno za začetnike)
- **Cursor** (Urejevalnik kode, podprt z AI)
- **Windsurf by Codeium** (Platforma za razvoj AI)
- Druga orodja, združljiva z MCP

### Korak 2: Poiščite svojo konfiguracijsko datoteko

Glede na vaše AI orodje boste morali urediti eno od teh datotek:

- **Claude Desktop**: `~/Library/Application\ Support/Claude/claude_desktop_config.json` za macOS in `%APPDATA%\Claude\claude_desktop_config.json` za Windows
- **Cursor**: `.cursor/mcp.json`
- **Windsurf**: `.codeium/windsurf/mcp_config.json`
- **Druga orodja**: Preverite MCP dokumentacijo vašega orodja

### Korak 3: Dodajte Ultracite konfiguracijo

Kopirajte in prilepite to konfiguracijo v vašo MCP konfiguracijsko datoteko:

```json
{
  "mcpServers": {
    "ultracite": {
      "command": "npx",
      "args": [
        "-y",
        "mcp-remote",
        "https://www.ultracite.ai/api/mcp/mcp"
      ]
    }
  }
}
```

### Korak 4: Znova zaženite vaše AI orodje

Zaprite in znova odprite vašo AI aplikacijo, da spremembe stopijo v veljavo.

### Korak 5: Preverite povezavo

Preizkusite integracijo tako, da vprašate vašega AI asistenta:

> Katera pravila Ultracite so na voljo?

Če uspe, bo vaš AI lahko navedel in pojasnil Ultracite pravila!

## Več MCP strežnikov

Uporabljate lahko Ultracite skupaj z drugimi MCP strežniki:

```json
{
  "mcpServers": {
    "ultracite": {
      "command": "npx",
      "args": ["-y", "mcp-remote", "https://www.ultracite.ai/api/mcp/mcp"]
    },
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"]
    },
    "filesystem": {
      "command": "npx", 
      "args": ["-y", "@modelcontextprotocol/server-filesystem", "/path/to/project"]
    }
  }
}
```

## Primeri uporabe

### Raziščite razpoložljiva pravila

Vprašajte vašega AI asistenta:

> Katera pravila ima Ultracite za React hooke?

Pričakovani odgovor bo vključeval ustrezna pravila za lintanje React hookov in njihove opise.

### Razumevanje kategorij pravil

> Pokaži mi vsa pravila za dostopnost v Ultracite

Vaš AI lahko našteje in pojasni pravila dostopnosti (a11y), konfigurirana v Ultracite.

### Primerjava prednastavitev

> Kakšna je razlika med ultracite/core in ultracite/react?

Pridobite podrobne informacije o različnih prednastavitvah konfiguracij in kdaj jih uporabiti.

## Varnost in zasebnost

### Ravnanje z podatki

- Ultracite MCP strežnik zagotavlja le javno dokumentacijo pravil za lintanje in informacije o konfiguraciji
- Nobeni osebni podatki ali koda niso preneseni na naše strežnike
- Vsa komunikacija poteka skozi varnostni sloj izbranega AI orodja

### Avtentikacija

- Za dostop do javne dokumentacije avtentikacija ni potrebna
- Prihodnje plačljive funkcije bodo morda zahtevale API ključe
- Vedno uporabljajte uradne Ultracite MCP končne točke
