Ultracite, monorepolarla kutudan çıktığı gibi çalışacak şekilde tasarlanmıştır. İşte Turborepo ile nasıl kullanacağınız.

## Yapılandırma

Ultracite'i başlatırken, depo kökünde bir `biome.jsonc` dosyası oluşturulacaktır. Bu dosya Ultracite'i tüm monoreponuz için yapılandırmakta kullanılacaktır.

```
my-turborepo/
├── apps
├── packages
├── turbo.json
└── biome.jsonc
```

## Betikler ekleme

Biome çok hızlı olduğu için, paketlerinizin her birinde ayrı betikler oluşturmak yerine kök bir betik kullanabilirsiniz.

`package.json` dosyanıza `check` ve `fix` betiklerini ekleyebilirsiniz.

```json title="package.json"
{
  "scripts": {
    "check": "ultracite check",
    "fix": "ultracite fix"
  }
}
```

İsteğe bağlı olarak `check` ve `fix` betiklerini `turbo.json` dosyanıza da ekleyebilirsiniz.

```json title="turbo.json"
{
  "tasks": {
    "//#check": {},
    "//#fix": {
      "cache": false
    }
  }
}
```

Artık bu betikleri `turbo run check` ve `turbo run fix` kullanarak çalıştırabilirsiniz.
