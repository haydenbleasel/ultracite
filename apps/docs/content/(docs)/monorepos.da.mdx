title: Monorepoer

---

Ultracite er designet til at fungere direkte med monorepos. Sådan bruger du det med Turborepo.

## Konfiguration

Når du initialiserer Ultracite, vil det oprette en `biome.jsonc`-fil i roden af dit repository. Denne fil vil blive brugt til at konfigurere Ultracite for hele dit monorepo.

```
my-turborepo/
├── apps
├── packages
├── turbo.json
└── biome.jsonc
```

## Tilføjelse af scripts

Fordi Biome er så hurtigt, kan du bruge et root-script i stedet for at oprette separate scripts i hver af dine pakker.

Du kan tilføje `check`- og `fix`-scripts til din `package.json`-fil.

```json title="package.json"
{
  "scripts": {
    "check": "ultracite check",
    "fix": "ultracite fix"
  }
}
```

Valgfrit kan du også tilføje `check`- og `fix`-scripts til din `turbo.json`-fil.

```json title="turbo.json"
{
  "tasks": {
    "//#check": {},
    "//#fix": {
      "cache": false
    }
  }
}
```

Du kan nu køre disse scripts med `turbo run check` og `turbo run fix`.
