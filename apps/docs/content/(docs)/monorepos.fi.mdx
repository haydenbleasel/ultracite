Ultracite on suunniteltu toimimaan suoraan monorepojen kanssa. Tässä ohjeet sen käyttämiseen Turborepon kanssa.

## Konfigurointi

Alustaessasi Ultracitea se luo `biome.jsonc`-tiedoston repositoriosi juureen. Tätä tiedostoa käytetään konfiguroimaan Ultracite koko monorepollesi.

```
my-turborepo/
├── apps
├── packages
├── turbo.json
└── biome.jsonc
```

## Skriptien lisääminen

Koska Biome on niin nopea, voit käyttää juuritason skriptiä sen sijaan, että loisit erilliset skriptit jokaiseen pakettiisi.

Voit lisätä `check`- ja `fix`-skriptit `package.json`-tiedostoosi.

```json title="package.json"
{
  "scripts": {
    "check": "ultracite check",
    "fix": "ultracite fix"
  }
}
```

Valinnaisesti voit myös lisätä `check`- ja `fix`-skriptit `turbo.json`-tiedostoosi.

```json title="turbo.json"
{
  "tasks": {
    "//#check": {},
    "//#fix": {
      "cache": false
    }
  }
}
```

Nyt voit suorittaa nämä skriptit komentoilla `turbo run check` ja `turbo run fix`.
