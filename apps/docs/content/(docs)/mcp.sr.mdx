The **Protokol za kontekst modela (MCP)** je otvoreni standard koji omogućava AI asistentima kao što su Claude, Cursor i drugim alatima pokretanim veštačkom inteligencijom da se bezbedno povežu sa spoljnim izvorima podataka i sistemima. Zamislite ga kao "univerzalni daljinski" koji omogućava vašim AI alatima pristup stvarnim podacima i funkcionalnostima.

Ultracite podržava MCP kako bi unapredio vaš razvojni tok rada za AI.

## Vodič za instalaciju

### Korak 1: Izaberite svoj AI alat

Prvo, uverite se da koristite alat za razvoj AI koji podržava MCP:

- [Claude Desktop](https://claude.ai/download) (Besplatno - preporučeno za početnike)
- [Cursor](https://www.cursor.com/) (AI-pokretani uređivač koda)
- [Windsurf by Codeium](https://windsurf.com/) (AI razvojna platforma)
- Ostali alati kompatibilni sa MCP

### Korak 2: Pronađite svoju konfiguracionu datoteku

U zavisnosti od vašeg AI alata, biće potrebno da uredite jednu od ovih datoteka:

- **Claude Desktop**: `~/Library/Application\ Support/Claude/claude_desktop_config.json` za macOS i `%APPDATA%\Claude\claude_desktop_config.json` za Windows
- **Cursor**: `.cursor/mcp.json`
- **Windsurf**: `.codeium/windsurf/mcp_config.json`
- **Ostali alati**: Proverite MCP dokumentaciju vašeg alata

### Korak 3: Dodajte Ultracite konfiguraciju

Kopirajte i nalepite ovu konfiguraciju u vašu MCP konfiguracionu datoteku:

```json
{
  "mcpServers": {
    "ultracite": {
      "command": "npx",
      "args": [
        "-y",
        "mcp-remote",
        "https://www.ultracite.ai/api/mcp/mcp"
      ]
    }
  }
}
```

### Korak 4: Ponovo pokrenite svoj AI alat

Zatvorite i ponovo otvorite vašu AI aplikaciju da bi izmene stupile na snagu.

### Korak 5: Proverite vezu

Testirajte integraciju tako što ćete pitati svog AI asistenta:

> Koja Ultracite pravila su dostupna?

Ako je uspešno, vaš AI bi trebalo da može da navede i objasni Ultracite pravila!

## Više MCP servera

Možete koristiti Ultracite zajedno sa drugim MCP serverima:

```json
{
  "mcpServers": {
    "ultracite": {
      "command": "npx",
      "args": ["-y", "mcp-remote", "https://www.ultracite.ai/api/mcp/mcp"]
    },
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"]
    },
    "filesystem": {
      "command": "npx", 
      "args": ["-y", "@modelcontextprotocol/server-filesystem", "/path/to/project"]
    }
  }
}
```

## Primeri upotrebe

### Istraživanje dostupnih pravila

Pitajte svog AI asistenta:

> Koja pravila Ultracite ima za React hooks?

Očekivani odgovor treba da uključi relevantna linting pravila za React hook-ove i njihove opise.

### Razumevanje kategorija pravila

> Pokaži mi sva pravila pristupačnosti u Ultracite

Vaš AI može navesti i objasniti a11y pravila konfigurisana u Ultracite.

### Poređenje preddefinisanih konfiguracija

> Koja je razlika između ultracite/core i ultracite/react?

Dobijte detaljne informacije o različitim preddefinisanim konfiguracijama i kada ih koristiti.

## Bezbednost i privatnost

### Rukovanje podacima

- Ultracite MCP server pruža samo javnu dokumentaciju o linting pravilima i informacije o konfiguraciji
- Nijedan lični podatak ili kod se ne prenosi na naše servere
- Sva komunikacija se odvija kroz bezbednosni sloj vašeg izabranog AI alata

### Autentifikacija

- Za pristup javnoj dokumentaciji nije potrebna autentifikacija
- Buduće premium funkcije mogu zahtevati API ključeve
- Uvek koristite zvanične Ultracite MCP endpoint-e
