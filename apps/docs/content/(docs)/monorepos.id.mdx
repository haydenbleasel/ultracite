Ultracite dirancang untuk bekerja secara langsung dengan monorepo. Berikut cara menggunakannya dengan Turborepo.

## Konfigurasi

Saat menginisialisasi Ultracite, ia akan membuat file `biome.jsonc` di root repositori Anda. File ini akan digunakan untuk mengonfigurasi Ultracite untuk seluruh monorepo Anda.

```
my-turborepo/
├── apps
├── packages
├── turbo.json
└── biome.jsonc
```

## Menambahkan skrip

Karena Biome sangat cepat, Anda dapat menggunakan skrip di root alih-alih membuat skrip terpisah di setiap paket Anda.

Anda dapat menambahkan skrip `check` dan `fix` ke file `package.json` Anda.

```json title="package.json"
{
  "scripts": {
    "check": "ultracite check",
    "fix": "ultracite fix"
  }
}
```

Opsional, Anda juga dapat menambahkan skrip `check` dan `fix` ke file `turbo.json` Anda.

```json title="turbo.json"
{
  "tasks": {
    "//#check": {},
    "//#fix": {
      "cache": false
    }
  }
}
```

Sekarang Anda dapat menjalankan skrip ini menggunakan `turbo run check` dan `turbo run fix`.
