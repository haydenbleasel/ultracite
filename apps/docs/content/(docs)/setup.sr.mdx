Ultracite se može instalirati pomoću svih glavnih package manager-a. Potreban vam je Node.js v14.18+ (preporučeno Node 18+) i projekat koji sadrži fajl `package.json`.

## Instalacija

Najlakši način da počnete je da pokrenete Ultracite-ovu `init` skriptu, koja će instalirati neophodne pakete i podesiti config fajl umesto vas. U direktorijumu vašeg projekta pokrenite:

```package-install
npx ultracite init
```

Opcionalno možete proslediti sledeće opcije komandi `init`:

| Zastavica            | Opis                                             | Opcije                                                                  |
|----------------------|--------------------------------------------------|-------------------------------------------------------------------------|
| `--pm`               | Paket menadžer koji će se koristiti.             | `pnpm`, `bun`, `yarn`, `npm`                                            |
| `--frameworks`       | Framework-i koji će biti konfigurisani.          | Odvojeno zarezom: `react`, `next`, `solid`, `vue`, `qwik`, `angular`, `remix`, `svelte` |
| `--editors`          | Editori koji će biti konfigurisani.              | Odvojeno zarezom: `vscode`, `zed`                                        |
| `--agents`           | AI agenti koji će biti konfigurisani.            | Odvojeno zarezom: `vscode-copilot`, `cursor`, `windsurf`, `zed`, `claude`, `codex`, `kiro`, `cline`, `amp`, `aider`, `firebase-studio`, `open-hands`, `gemini-cli`, `junie`, `augmentcode`, `kilo-code`, `goose`, `roo-code`, `warp` |
| `--hooks`            | Agent hook-ovi koji će biti konfigurisani.       | Odvojeno zarezom: `cursor`, `claude`                                     |
| `--integrations`     | Dodatne integracije za podešavanje.              | Odvojeno zarezom: `husky`, `lefthook`, `lint-staged`                     |
| `--migrate`          | Alati za migraciju koje treba ukloniti.          | Odvojeno zarezom: `eslint`, `prettier` (uklanja zavisnosti, konfiguracione fajlove i podešavanja editora) |
| `--skip-install`     | Preskoči instalaciju zavisnosti.                 | (nije potrebna vrednost)                                                |
| `--quiet`            | Sakrij interaktivne upite i vizuelni izlaz.      | (nije potrebna vrednost, automatski omogućeno u CI)                      |

### Programatsko korišćenje

Za alate koji moraju da pozivaju Ultracite programatski (npr. scaffolding CLI alati), možete koristiti zastavicu `--quiet` da potisnete sve interaktivne upite i vizuelni izlaz:

```bash title="Terminal"
npx ultracite init --quiet --pm npm
```

Zastavica `--quiet` se automatski omogućava kada je promenljiva okruženja `CI` postavljena na `true` ili `1`, što je čini pogodnom za CI/CD pipeline-ove:

```bash title="Terminal"
CI=true npx ultracite init --pm npm
```

U tihom režimu, Ultracite će:
- Preskočiti sve interaktivne upite
- Koristiti razumna podrazumevana podešavanja za sve opcije
- Potisnuti ASCII art, spanere i log poruke
- Kreirati samo osnovnu Biome konfiguraciju
- Izaći uredno sa kodom 0 pri uspehu ili 1 pri neuspehu

## Ekstenzije

Ako koristite Visual Studio Code, instalirajte zvaničnu [Biome](https://marketplace.visualstudio.com/items?itemName=biomejs.biome) ekstenziju, koja omogućava formatiranje/otklanjanje grešaka pri čuvanju:

```bash title="Terminal"
code --install-extension biomejs.biome
```

Pored toga, ako koristite Tailwind CSS, preporučuje se instalacija [Tailwind CSS IntelliSense](https://marketplace.visualstudio.com/items?itemName=bradlc.vscode-tailwindcss) ekstenzije za sortiranje klasa i sugestije:

```bash title="Terminal"
code --install-extension bradlc.vscode-tailwindcss
```
