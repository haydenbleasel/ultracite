Ultracite, düzenleyiciye özel kural dosyalarını otomatik olarak üreterek yapay zeka kodlama asistanlarıyla sorunsuz entegrasyon sağlar. Bu kurallar, yapay zeka kodlama asistanlarını en iyi uygulamalara uyan, daha tutarlı ve daha iyi kod yazmaları için yönlendirir.

Editör kuralları, Biome'un linting ve biçimlendirmesi ile birlikte çalışır. Biome kod biçimlendirmesini yönetirken ve sözdizimi hatalarını yakalarken, editör kuralları yapay zeka kodlama asistanlarını baştan daha iyi kod yazmaları için yönlendirir ve sonradan düzeltme ihtiyacını azaltır.

Bu çift yaklaşım hem otomatik kod kalitesini (Biome aracılığıyla) hem de akıllı kod üretimini (yapay zeka kod kuralları aracılığıyla) garanti eder.

## Kurulum

### Otomatik Kurulum

When you run `npx ultracite init`, you'll be prompted to select which editor rules you want to enable:

```bash
? Which editor rules do you want to enable (optional)?
  ◯ GitHub Copilot (VSCode)
  ◯ Cursor
  ◯ Windsurf
  # ... etcetera
```

This will create the relevant file in your project directory.

### Manuel Kurulum

If you want to manually set things up, create the relevant file in your project directory (for example `.cursor/rules` for Cursor or `.rules` for Zed). Then, copy and paste [this content](https://github.com/haydenbleasel/ultracite/tree/main/.cursor/rules/ultracite.mdc) into the file.

## Faydalar

Ultracite editör kurallarını kullanmanın birkaç avantajı vardır:

- **Tutarlılık**: Tüm yapay zeka tarafından üretilen kod aynı konvansiyonlara uyar
- **Kalite**: Kurallar en iyi uygulamaları zorunlu kılar ve yaygın hataları önler
- **Erişilebilirlik**: Dahili erişilebilirlik yönergeleri kapsayıcı kod sağlar
- **Performans**: Kurallar verimli kalıpları teşvik eder ve anti-pattern'lerden kaçınır
- **Güvenlik**: Yönergeler yaygın güvenlik açıklarını önlemeye yardımcı olur

## Özelleştirme

Varsayılan kurallar kapsamlı olsa da, bunları özel ihtiyaçlarınıza göre özelleştirebilirsiniz:

1. **Mevcut kuralları değiştirin**: Kural dosyalarını doğrudan düzenleyin
2. **Proje-özel kurallar ekleyin**: Ek yönergeler ekleyin
3. **Kuralları devre dışı bırakın**: Projeniz için geçerli olmayan kuralları kaldırın veya yorum satırı haline getirin
