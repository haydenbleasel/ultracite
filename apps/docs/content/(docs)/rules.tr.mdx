Ultracite, düzenleyiciye özgü kural dosyalarını otomatik olarak oluşturarak yapay zeka kodlama asistanlarıyla sorunsuz entegrasyon sağlar. Bu kurallar, yapay zeka kodlama asistanlarının en iyi uygulamalara uyan, daha tutarlı ve daha kaliteli kod yazmasına yardımcı olur.

Düzenleyici kuralları, Biome'un linting ve biçimlendirmesiyle birlikte çalışır. Biome kod biçimlendirmesini sağlar ve sözdizimi hatalarını yakalarken, düzenleyici kuralları yapay zeka kodlama asistanlarını baştan daha iyi kod yazmaları için yönlendirir ve sonradan düzeltme gereksinimini azaltır.

Bu çift yönlü yaklaşım hem otomatik kod kalitesini (Biome aracılığıyla) hem de akıllı kod üretimini (yapay zeka kodlama kuralları aracılığıyla) garanti eder.

## Kurulum

### Otomatik Kurulum

When you run `npx ultracite init`, you'll be prompted to select which editor rules you want to enable:

```bash
? Which editor rules do you want to enable (optional)?
  ◯ GitHub Copilot (VSCode)
  ◯ Cursor
  ◯ Windsurf
  # ... etcetera
```

Bu, proje dizininizde ilgili dosyayı oluşturacaktır.

### Manuel Kurulum

If you want to manually set things up, create the relevant file in your project directory (for example `.cursor/rules` for Cursor or `.rules` for Zed). Then, copy and paste [bu içeriği](https://github.com/haydenbleasel/ultracite/tree/main/.cursor/rules/ultracite.mdc) into the file.

## Faydalar

Ultracite düzenleyici kurallarını kullanmanın çeşitli avantajları vardır:

- **Tutarlılık**: Tüm yapay zeka tarafından üretilen kod aynı konvansiyonları takip eder
- **Kalite**: Kurallar en iyi uygulamaları uygular ve yaygın hataları önler
- **Erişilebilirlik**: Yerleşik erişilebilirlik yönergeleri kapsayıcı kod sağlamayı garanti eder
- **Performans**: Kurallar verimli desenleri teşvik eder ve anti-desenlerden kaçınır
- **Güvenlik**: Yönergeler yaygın güvenlik açıklarının önlenmesine yardımcı olur

## Özelleştirme

Varsayılan kurallar kapsamlı olsa da, özel ihtiyaçlarınıza göre bunları özelleştirebilirsiniz:

1. **Mevcut kuralları düzenleyin**: Kural dosyalarını doğrudan düzenleyin
2. **Proje-özgü kurallar ekleyin**: Ek yönergeler ekleyin
3. **Kuralları devre dışı bırakın**: Projenize uymayan kuralları kaldırın veya yorum satırı yapın
