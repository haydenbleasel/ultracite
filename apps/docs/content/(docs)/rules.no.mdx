Ultracite tilbyr sømløs integrasjon med AI-kodeassistenter ved automatisk å generere editor-spesifikke regelfiler. Disse reglene hjelper til med å veilede AI-kodeassistenter til å skrive bedre, mer konsistent kode som følger beste praksis.

Editorreglene fungerer sammen med Biome's linting og formatering. Mens Biome håndterer kodeformatering og fanger syntaksfeil, veileder editorreglene AI-kodeassistenter til å skrive bedre kode fra starten av, noe som reduserer behovet for rettelser senere.

Denne dobbeltilnærmingen sikrer både automatisert kodekvalitet (via Biome) og intelligent kodegenerering (gjennom regler for AI-koding).

## Oppsett

### Automatisk oppsett

When you run `npx ultracite init`, you'll be prompted to select which editor rules you want to enable:

```bash
? Which editor rules do you want to enable (optional)?
  ◯ GitHub Copilot (VSCode)
  ◯ Cursor
  ◯ Windsurf
  # ... etcetera
```

Dette vil opprette den relevante filen i prosjektmappen din.

### Manuelt oppsett

Hvis du vil sette opp ting manuelt, opprett den relevante filen i prosjektmappen din (for eksempel `.cursor/rules` for Cursor eller `.rules` for Zed). Deretter kopierer og limer du inn [dette innholdet](https://github.com/haydenbleasel/ultracite/tree/main/.cursor/rules/ultracite.mdc) i filen.

## Fordeler

Bruk av Ultracite-editorregler gir flere fordeler:

- **Konsistens**: All AI-generert kode følger de samme konvensjonene
- **Kvalitet**: Regler håndhever beste praksis og forhindrer vanlige feil
- **Tilgjengelighet**: Innebygde retningslinjer for tilgjengelighet sikrer inkluderende kode
- **Ytelse**: Regler fremmer effektive mønstre og unngår anti-mønstre
- **Sikkerhet**: Retningslinjer bidrar til å forhindre vanlige sikkerhetssårbarheter

## Tilpasning

Mens standardreglene er omfattende, kan du tilpasse dem for dine spesifikke behov:

1. **Endre eksisterende regler**: Rediger regelfilene direkte
2. **Legg til prosjektspesifikke regler**: Legg til ekstra retningslinjer
3. **Deaktiver regler**: Fjern eller kommenter ut regler som ikke gjelder for prosjektet ditt
