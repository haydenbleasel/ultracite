Het **Model Context Protocol (MCP)** is een open standaard die AI-assistenten zoals Claude, Cursor en andere AI-aangedreven tools in staat stelt om veilig verbinding te maken met externe gegevensbronnen en systemen. Zie het als een "universele afstandsbediening" waarmee uw AI-tools toegang krijgen tot gegevens uit de echte wereld en functionaliteit.

Ultracite ondersteunt MCP om uw AI-ontwikkelworkflow te versterken.

## Installatiegids

### Stap 1: Kies uw AI-tool

Zorg er eerst voor dat u een AI-ontwikkeltool gebruikt die MCP ondersteunt:

- [Claude Desktop](https://claude.ai/download) (Gratis - aanbevolen voor beginners)
- [Cursor](https://www.cursor.com/) (AI-aangedreven code-editor)
- [Windsurf by Codeium](https://windsurf.com/) (AI-ontwikkelplatform)
- Andere MCP-compatibele tools

### Stap 2: Vind uw configuratiebestand

Afhankelijk van uw AI-tool moet u een van deze bestanden bewerken:

- **Claude Desktop**: `~/Library/Application\ Support/Claude/claude_desktop_config.json` voor macOS en `%APPDATA%\Claude\claude_desktop_config.json` voor Windows
- **Cursor**: `.cursor/mcp.json`
- **Windsurf**: `.codeium/windsurf/mcp_config.json`
- **Andere tools**: Raadpleeg de MCP-documentatie van uw tool

### Stap 3: Voeg Ultracite-configuratie toe

Kopieer en plak deze configuratie in uw MCP-configuratiebestand:

```json
{
  "mcpServers": {
    "ultracite": {
      "command": "npx",
      "args": [
        "-y",
        "mcp-remote",
        "https://www.ultracite.ai/api/mcp/mcp"
      ]
    }
  }
}
```

### Stap 4: Start uw AI-tool opnieuw

Sluit en open uw AI-applicatie opnieuw zodat de wijzigingen van kracht worden.

### Stap 5: Controleer de verbinding

Test de integratie door uw AI-assistent te vragen:

> Welke Ultracite-regels zijn beschikbaar?

Als het lukt, zou uw AI in staat moeten zijn om Ultracite-regels op te sommen en uit te leggen!

## Meerdere MCP-servers

U kunt Ultracite naast andere MCP-servers gebruiken:

```json
{
  "mcpServers": {
    "ultracite": {
      "command": "npx",
      "args": ["-y", "mcp-remote", "https://www.ultracite.ai/api/mcp/mcp"]
    },
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"]
    },
    "filesystem": {
      "command": "npx", 
      "args": ["-y", "@modelcontextprotocol/server-filesystem", "/path/to/project"]
    }
  }
}
```

## Gebruiksvoorbeelden

### Beschikbare regels verkennen

Vraag uw AI-assistent:

> Welke regels heeft Ultracite voor React-hooks?

Het verwachte antwoord bevat relevante lintregels voor React-hooks en hun beschrijvingen.

### Regelcategorieën begrijpen

> Toon alle toegankelijkheidsregels in Ultracite

Uw AI kan de a11y-regels die in Ultracite zijn geconfigureerd, opsommen en uitleggen.

### Voorinstellingen vergelijken

> Wat is het verschil tussen ultracite/core en ultracite/react?

Krijg gedetailleerde informatie over verschillende presetconfiguraties en wanneer u ze moet gebruiken.

## Beveiliging en privacy

### Gegevensverwerking

- De Ultracite-MCP-server verstrekt alleen openbare documentatie over lintregels en configuratie-informatie
- Er worden geen persoonlijke gegevens of code naar onze servers verzonden
- Alle communicatie verloopt via de beveiligingslaag van uw gekozen AI-tool

### Authenticatie

- Geen authenticatie vereist voor toegang tot openbare documentatie
- Toekomstige premiumfuncties kunnen API-sleutels vereisen
- Gebruik altijd officiële Ultracite MCP-endpoints
