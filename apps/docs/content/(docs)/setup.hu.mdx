Ultracite telepíthető minden jelentősebb csomagkezelővel. Szükséged lesz Node.js v14.18+-ra (ajánlott a Node 18+) és egy projektre, amely tartalmaz egy `package.json` fájlt.

## Telepítés

A legegyszerűbb mód a kezdéshez az, ha futtatod az Ultracite `init` szkriptjét, amely telepíti a szükséges csomagokat és beállítja a konfigurációs fájlt helyetted. A projekt könyvtáradban futtasd:

```package-install
npx ultracite init
```

Opcionálisan az `init` parancshoz megadhatod a következő zászlókat:

| Kapcsoló             | Leírás                                           | Opciók                                                                  |
|----------------------|--------------------------------------------------|-------------------------------------------------------------------------|
| `--pm`               | A használni kívánt csomagkezelő.                 | `pnpm`, `bun`, `yarn`, `npm`                                            |
| `--frameworks`       | A konfigurálandó keretrendszerek.                | Vesszővel elválasztva: `react`, `next`, `solid`, `vue`, `qwik`, `angular`, `remix`, `svelte` |
| `--editors`          | A konfigurálandó szerkesztők.                    | Vesszővel elválasztva: `vscode`, `zed`                                   |
| `--agents`           | A konfigurálandó AI ügynökök.                    | Vesszővel elválasztva: `vscode-copilot`, `cursor`, `windsurf`, `zed`, `claude`, `codex`, `kiro`, `cline`, `amp`, `aider`, `firebase-studio`, `open-hands`, `gemini-cli`, `junie`, `augmentcode`, `kilo-code`, `goose`, `roo-code`, `warp`, `droid` |
| `--hooks`            | A konfigurálandó ügynök hookok.                  | Vesszővel elválasztva: `cursor`, `claude`                                |
| `--integrations`     | További beállítandó integrációk.                 | Vesszővel elválasztva: `husky`, `lefthook`, `lint-staged`                |
| `--migrate`          | Eltávolítandó migrációs eszközök.                | Vesszővel elválasztva: `eslint`, `prettier` (eltávolítja a függőségeket, konfigurációs fájlokat és a szerkesztőbeállításokat) |
| `--skip-install`     | A függőségek telepítésének kihagyása.            | (érték megadása nem szükséges)                                          |
| `--quiet`            | Interaktív kérdések és vizuális kimenet elnyomása.| (érték megadása nem szükséges, CI-ben automatikusan engedélyezett)      |

### Programozott használat

Olyan eszközök számára, amelyek programból hívják az Ultracite-et (például scaffolding CLI-k), használhatod a `--quiet` kapcsolót az összes interaktív kérdés és vizuális kimenet elnyomására:

```bash title="Terminal"
npx ultracite init --quiet --pm npm
```

A `--quiet` kapcsoló automatikusan engedélyezve van, amikor a `CI` környezeti változó `true`-ra vagy `1`-re van állítva, így CI/CD folyamatokhoz is alkalmas:

```bash title="Terminal"
CI=true npx ultracite init --pm npm
```

Csendes módban az Ultracite a következőket fogja tenni:
- Kihagyja az összes interaktív kérdést
- Ésszerű alapértelmezéseket használ minden opció esetén
- Elnyomja az ASCII grafikákat, a forgó indikátorokat és a naplóüzeneteket
- Csak az alapvető Biome konfigurációt hozza létre
- Siker esetén 0-val, hiba esetén 1-gyel lép ki

## Bővítmények

Ha Visual Studio Code-ot használsz, telepítsd a hivatalos [Biome](https://marketplace.visualstudio.com/items?itemName=biomejs.biome) bővítményt, amely lehetővé teszi a mentéskor történő formázást/hibajavítást:

```bash title="Terminal"
code --install-extension biomejs.biome
```

Ezen felül, ha Tailwind CSS-t használsz, ajánlott telepíteni a [Tailwind CSS IntelliSense](https://marketplace.visualstudio.com/items?itemName=bradlc.vscode-tailwindcss) bővítményt az osztályok rendezéséhez és javaslatokhoz:

```bash title="Terminal"
code --install-extension bradlc.vscode-tailwindcss
```
