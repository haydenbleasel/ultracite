Ultracite alapból támogatja a monorepókat. Íme, hogyan használhatod Turborepo-val.

## Konfiguráció

Az Ultracite inicializálásakor a repód gyökerébe létrehoz egy `biome.jsonc` fájlt. Ezt a fájlt az Ultracite egész monorepód konfigurálásához fogja használni.

```
my-turborepo/
├── apps
├── packages
├── turbo.json
└── biome.jsonc
```

## Szkriptek hozzáadása

Mivel a Biome olyan gyors, használhatsz egy gyökérszkriptet ahelyett, hogy minden csomagodban külön szkripteket hoznál létre.

Hozzáadhatod a `check` és `fix` szkripteket a `package.json` fájlodhoz.

```json title="package.json"
{
  "scripts": {
    "check": "ultracite check",
    "fix": "ultracite fix"
  }
}
```

Opcionálisan a `check` és `fix` szkripteket a `turbo.json` fájlhoz is hozzáadhatod.

```json title="turbo.json"
{
  "tasks": {
    "//#check": {},
    "//#fix": {
      "cache": false
    }
  }
}
```

Mostantól futtathatod ezeket a szkripteket a `turbo run check` és `turbo run fix` parancsokkal.
