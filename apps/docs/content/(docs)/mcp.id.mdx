The **Protokol Konteks Model (MCP)** adalah standar terbuka yang memungkinkan asisten AI seperti Claude, Cursor, dan alat bertenaga AI lainnya untuk terhubung secara aman dengan sumber data dan sistem eksternal. Anggaplah ini sebagai "remote universal" yang memungkinkan alat AI Anda mengakses data dan fungsi dunia nyata.

Ultracite mendukung MCP untuk meningkatkan alur kerja pengembangan AI Anda.

## Panduan Instalasi

### Langkah 1: Pilih Alat AI Anda

Pertama, pastikan Anda menggunakan alat pengembangan AI yang mendukung MCP:

- [Claude Desktop](https://claude.ai/download) (Gratis - direkomendasikan untuk pemula)
- [Cursor](https://www.cursor.com/) (editor kode bertenaga AI)
- [Windsurf by Codeium](https://windsurf.com/) (platform pengembangan AI)
- Alat lain yang kompatibel dengan MCP

### Langkah 2: Temukan File Konfigurasi Anda

Tergantung pada alat AI Anda, Anda perlu mengedit salah satu file ini:

- **Claude Desktop**: `~/Library/Application\ Support/Claude/claude_desktop_config.json` untuk macOS dan `%APPDATA%\Claude\claude_desktop_config.json` untuk Windows
- **Cursor**: `.cursor/mcp.json`
- **Windsurf**: `.codeium/windsurf/mcp_config.json`
- **Alat lain**: Periksa dokumentasi MCP alat Anda

### Langkah 3: Tambahkan Konfigurasi Ultracite

Salin dan tempel konfigurasi ini ke file konfigurasi MCP Anda:

```json
{
  "mcpServers": {
    "ultracite": {
      "command": "npx",
      "args": [
        "-y",
        "mcp-remote",
        "https://www.ultracite.ai/api/mcp/mcp"
      ]
    }
  }
}
```

### Langkah 4: Mulai Ulang Alat AI Anda

Tutup dan buka kembali aplikasi AI Anda agar perubahan berlaku.

### Langkah 5: Verifikasi Koneksi

Uji integrasi dengan menanyakan kepada asisten AI Anda:

> Aturan Ultracite apa saja yang tersedia?

Jika berhasil, AI Anda seharusnya dapat mendaftar dan menjelaskan aturan Ultracite!

## Beberapa Server MCP

Anda dapat menggunakan Ultracite bersamaan dengan server MCP lainnya:

```json
{
  "mcpServers": {
    "ultracite": {
      "command": "npx",
      "args": ["-y", "mcp-remote", "https://www.ultracite.ai/api/mcp/mcp"]
    },
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"]
    },
    "filesystem": {
      "command": "npx", 
      "args": ["-y", "@modelcontextprotocol/server-filesystem", "/path/to/project"]
    }
  }
}
```

## Contoh Penggunaan

### Menjelajahi Aturan yang Tersedia

Tanyakan kepada asisten AI Anda:

> Aturan apa yang dimiliki Ultracite untuk React hooks?

Respons yang diharapkan akan mencakup aturan linting React hook yang relevan dan deskripsinya.

### Memahami Kategori Aturan

> Tunjukkan semua aturan aksesibilitas di Ultracite

AI Anda dapat mendaftar dan menjelaskan aturan aksesibilitas (a11y) yang dikonfigurasi di Ultracite.

### Membandingkan Preset

> Apa perbedaan antara ultracite/core dan ultracite/react?

Dapatkan informasi terperinci tentang berbagai konfigurasi preset dan kapan menggunakannya.

## Keamanan dan Privasi

### Penanganan Data

- Server MCP Ultracite hanya menyediakan dokumentasi aturan linting publik dan informasi konfigurasi
- Tidak ada data pribadi atau kode yang dikirim ke server kami
- Semua komunikasi terjadi melalui lapisan keamanan alat AI yang Anda pilih

### Autentikasi

- Tidak diperlukan autentikasi untuk mengakses dokumentasi publik
- Fitur premium di masa depan mungkin memerlukan kunci API
- Selalu gunakan endpoint MCP Ultracite resmi
