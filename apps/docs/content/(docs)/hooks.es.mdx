---
title: Ganchos
description: Cómo integrar Ultracite con asistentes de codificación IA mediante ganchos del editor.
---

Ultracite puede formatear y corregir código automáticamente después de que los asistentes de codificación IA realicen ediciones mediante ganchos del editor. Esto garantiza que todo el código generado por IA cumpla con los estándares de tu proyecto sin intervención manual.

Los ganchos funcionan junto con el linting y el formateo de Biome. Después de que un asistente de codificación IA edite un archivo, el gancho ejecuta automáticamente `npx ultracite fix` para formatear y corregir los problemas de linting que pueden autcorregirse. Este enfoque dual asegura tanto la generación de código inteligente (mediante reglas de IA) como la calidad automática del código (mediante ganchos).

> **Nota:** Los ganchos son independientes de las reglas de codificación IA. La bandera `--agents` configura las reglas de codificación IA, mientras que la bandera `--hooks` configura el formateo automático después de las ediciones de archivos.

## Configuración

### Configuración automática

When you run `npx ultracite init`, you'll be prompted to select which editor hooks you want to enable:

```bash
? Which agent hooks do you want to enable (optional)?
  ◯ Cursor
  ◯ Claude Code
```

This will create the relevant hook configuration file in your project directory.

### Configuración manual

If you want to manually set things up or add hooks after initialization, run:

```bash
npx ultracite init --hooks cursor,claude
```

Alternatively, you can manually create or edit the hook configuration files in your project directory and add the appropriate hook command.

## Editores compatibles

| Editor      | Hook File Location          | Documentation                                                           |
| ----------- | --------------------------- | ----------------------------------------------------------------------- |
| Cursor      | `.cursor/hooks.json`        | [Ganchos de Cursor](https://cursor.com/docs/agent/hooks)                    |
| Claude Code | `.claude/settings.json`     | [Ganchos de Claude Code](https://docs.claude.com/en/docs/claude-code/hooks) |

## Cómo funciona

1. **Ejecución automática**: Después de que el agente IA edite un archivo, el gancho ejecuta automáticamente `npx ultracite fix`
2. **Formateo de código**: Ultracite formatea el código editado según la configuración de Biome
3. **Correcciones de linting**: Los problemas de linting que se pueden autcorregir se resuelven automáticamente
4. **Integración transparente**: El proceso ocurre de forma transparente sin interrumpir tu flujo de trabajo

## Beneficios

Usar los ganchos de editor de Ultracite ofrece varias ventajas:

- **Consistencia**: Todo el código generado por IA se formatea automáticamente para coincidir con tu estilo
- **Calidad**: Los problemas de linting se corrigen inmediatamente después de la generación de código
- **Eficiencia**: No es necesario ejecutar manualmente formateadores o linters tras las ediciones de la IA
- **Estándares**: Garantiza que el código generado por IA siempre cumpla con los estándares del proyecto
- **Commits limpios**: El código está correctamente formateado antes incluso de que lo revises

## Personalización

Ultracite se fusiona inteligentemente con las configuraciones de ganchos existentes:

- **Preserva los ganchos existentes**: Todos los ganchos existentes y sus comandos permanecen intactos
- **Evita duplicados**: Si Ultracite ya está configurado, no se añadirá de nuevo
- **No destructivo**: Se mantiene la estructura de configuración de ganchos existente

También puedes personalizar el comportamiento del gancho modificando directamente los archivos de configuración de ganchos si necesitas comandos de formateo diferentes u otras acciones posteriores a la edición.
