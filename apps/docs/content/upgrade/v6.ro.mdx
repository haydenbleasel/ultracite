---
title: Actualizare la v6
description: Ghid pentru actualizarea de la Ultracite v5 la v6 cu noul sistem de preseturi.
---

Ultracite v6 introduce [preseturi specifice fiecărui framework](/configuration#framework-specific-configurations) care îți oferă mai mult control asupra regulilor care se aplică proiectului tău. **Această actualizare nu introduce modificări care rup compatibilitatea** pentru utilizatorii Next.js/React.

## Ce s-a schimbat

v6 introduce [preseturi specifice fiecărui framework](/configuration#framework-specific-configurations):

- [`ultracite/core`](/preset/core) - Reguli de bază pentru JavaScript/TypeScript
- [`ultracite/react`](/preset/react) - Reguli specifice React
- [`ultracite/next`](/preset/next) - Reguli specifice Next.js
- [`ultracite/solid`](/preset/solid) - Reguli Solid.js
- [`ultracite/vue`](/preset/vue) - Reguli Vue.js
- [`ultracite/svelte`](/preset/svelte) - Reguli Svelte
- [`ultracite/qwik`](/preset/qwik) - Reguli Qwik
- [`ultracite/angular`](/preset/angular) - Parser HTML pentru Angular
- [`ultracite/remix`](/preset/remix) - Suport pentru rutare bazată pe fișiere

## Migrare

### Proiecte Next.js/React

Actualizați `biome.jsonc` pentru a extinde preseturile corespunzătoare:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core", "ultracite/react", "ultracite/next"]
}
```

### Proiecte non-React

Comutați la presetul core dacă nu folosiți React:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core"]
}
```

### Alte framework-uri

Folosiți presetul potrivit pentru framework-ul vostru:

```jsonc title="biome.jsonc"
{
  // Solid.js
  "extends": ["ultracite/core", "ultracite/solid"]

  // Vue.js
  "extends": ["ultracite/core", "ultracite/vue"]

  // Svelte
  "extends": ["ultracite/core", "ultracite/svelte"]

  // Qwik
  "extends": ["ultracite/core", "ultracite/qwik"]

  // Angular
  "extends": ["ultracite/core", "ultracite/angular"]
}
```

### Proiecte multi-framework

Combinați preseturile după cum este necesar:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core", "ultracite/react", "ultracite/solid"]
}
```
