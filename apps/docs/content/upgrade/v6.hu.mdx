---
title: Frissítés v6-ra
description: Útmutató az Ultracite v5-ről v6-ra való frissítéshez az új preset-rendszerrel.
---

Ultracite v6 keretrendszer-specifikus preseteket vezet be, amelyek nagyobb kontrollt adnak arra vonatkozóan, hogy mely szabályok érvényesülnek a projektedben. **Ez a frissítés nem okoz visszafelé kompatibilitási problémát** a Next.js/React felhasználóknak.

## Mi változott

v6 bevezeti a [keretrendszer-specifikus preseteket](/configuration#framework-specific-configurations):

- [`ultracite/core`](/preset/core) - Alap JavaScript/TypeScript szabályok
- [`ultracite/react`](/preset/react) - React-specifikus szabályok
- [`ultracite/next`](/preset/next) - Next.js-specifikus szabályok
- [`ultracite/solid`](/preset/solid) - Solid.js szabályok
- [`ultracite/vue`](/preset/vue) - Vue.js szabályok
- [`ultracite/svelte`](/preset/svelte) - Svelte szabályok
- [`ultracite/qwik`](/preset/qwik) - Qwik szabályok
- [`ultracite/angular`](/preset/angular) - Angular HTML elemző
- [`ultracite/remix`](/preset/remix) - Fájl alapú útválasztás támogatása

## Migráció

### Next.js/React projektek

Frissítsd a `biome.jsonc` fájlodat, hogy kiterjeszd a megfelelő preseteket:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core", "ultracite/react", "ultracite/next"]
}
```

### Nem-React projektek

Váltson a core presetre, ha nem használ React-et:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core"]
}
```

### Egyéb keretrendszerek

Használd a megfelelő presetet a keretrendszeredhez:

```jsonc title="biome.jsonc"
{
  // Solid.js
  "extends": ["ultracite/core", "ultracite/solid"]

  // Vue.js
  "extends": ["ultracite/core", "ultracite/vue"]

  // Svelte
  "extends": ["ultracite/core", "ultracite/svelte"]

  // Qwik
  "extends": ["ultracite/core", "ultracite/qwik"]

  // Angular
  "extends": ["ultracite/core", "ultracite/angular"]
}
```

### Többkeretrendszeres projektek

Kombináld a preseteket szükség szerint:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core", "ultracite/react", "ultracite/solid"]
}
```
