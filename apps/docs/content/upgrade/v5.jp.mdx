title: v5へのアップグレード

---

Ultracite v5 は大規模なリリースであり、重要な改善点と破壊的変更が含まれます。このガイドは既存の v4 設定を v5 にアップグレードするのに役立ちます。

## v5 の新機能

- **更新された Biome**: v1.x から v2.x へアップグレード
- **新しい構成フォーマット**: `biome.json` から `biome.jsonc` へ切り替え
- **強化された CLI**: TypeScript による書き直しと改善されたエラー処理
- **エディタ統合**: Cursor、Windsurf、Zed、および GitHub Copilot の組み込みサポート
- **Git フック**: Husky、lint-staged、lefthook を含むオプションの統合
- **改善されたスキーマ**: より良い検証を備えた新しいスキーマ URL
- **簡素化された VS Code 設定**: ファイルタイプのオーバーライドを単一行に統合

## 移行手順

### 自動移行（推奨）

移行する最も簡単な方法は新しい初期化コマンドを実行することです:

```package-install
npx ultracite@latest init
```

これにより次のことが行われます:
- 依存関係を最新バージョンに更新
- `biome.json` または `biome.jsonc` を更新  
- VS Code 設定を更新
- AI エディタルール（Cursor、Windsurf、Zed など）の設定を提案
- 新しい統合（Git フック、lint-staged など）の設定を提案

アップグレード後、オーバーライドを削除するために `biome.json` / `biome.jsonc` ファイルを確認することをおすすめします。さらに、新しい構成が適用されるようにエディタを再起動してください。

また、スキーマ / ルール変更の詳細については Biome の [v2 アップグレードガイド](https://biomejs.dev/guides/upgrade-to-biome-v2/) を確認してください。

### 手動移行

より細かく制御したい場合は、[手動移行](/setup#manual-installation) の手順に従って次を行ってください:

- Ultracite と Biome の更新版を手動でインストールする
- `biome.json` ファイルを新しいスキーマバージョンに更新する
- `.vscode/settings.json` ファイルを更新する（オプション）
- AI エディタルールファイルを追加する（オプション）
- 新しい統合を追加する（オプション）

移行後、エディタを再起動して新しい設定が適用されていることを確認してください。
