---
title: Uppgradera till v6
description: Guide för uppgradering från Ultracite v5 till v6 med det nya preset-systemet.
---

Ultracite v6 inför ramverksspecifika presets som ger dig mer kontroll över vilka regler som gäller för ditt projekt. **Denna uppgradering är icke-brytande** för Next.js/React-användare.

## Vad som ändrades

v6 introducerar [ramverksspecifika presets](/configuration#framework-specific-configurations):

- [`ultracite/core`](/preset/core) - Basregler för JavaScript/TypeScript
- [`ultracite/react`](/preset/react) - React-specifika regler
- [`ultracite/next`](/preset/next) - Next.js-specifika regler
- [`ultracite/solid`](/preset/solid) - Solid.js-regler
- [`ultracite/vue`](/preset/vue) - Vue.js-regler
- [`ultracite/svelte`](/preset/svelte) - Svelte-regler
- [`ultracite/qwik`](/preset/qwik) - Qwik-regler
- [`ultracite/angular`](/preset/angular) - Angular HTML-parser
- [`ultracite/remix`](/preset/remix) - Stöd för filbaserad routing

## Migrering

### Next.js/React-projekt

Uppdatera din `biome.jsonc` så att den utökar lämpliga presets:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core", "ultracite/react", "ultracite/next"]
}
```

### Projekt utan React

Byt till core-presetet om du inte använder React:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core"]
}
```

### Andra ramverk

Använd det lämpliga presetet för ditt ramverk:

```jsonc title="biome.jsonc"
{
  // Solid.js
  "extends": ["ultracite/core", "ultracite/solid"]

  // Vue.js
  "extends": ["ultracite/core", "ultracite/vue"]

  // Svelte
  "extends": ["ultracite/core", "ultracite/svelte"]

  // Qwik
  "extends": ["ultracite/core", "ultracite/qwik"]

  // Angular
  "extends": ["ultracite/core", "ultracite/angular"]
}
```

### Projekt med flera ramverk

Kombinera presets efter behov:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core", "ultracite/react", "ultracite/solid"]
}
```
