---
title: Upgrade auf v6
description: Anleitung zum Upgrade von Ultracite v5 auf v6 mit dem neuen Preset-System.
---

Ultracite v6 führt [framework-spezifische Presets](/configuration#framework-specific-configurations) ein, die dir mehr Kontrolle darüber geben, welche Regeln auf dein Projekt angewendet werden. **Dieses Upgrade ist abwärtskompatibel** für Next.js/React-Benutzer.

## Was hat sich geändert

v6 führt [framework-spezifische Presets](/configuration#framework-specific-configurations) ein:

- [`ultracite/core`](/preset/core) - Basisregeln für JavaScript/TypeScript
- [`ultracite/react`](/preset/react) - React-spezifische Regeln
- [`ultracite/next`](/preset/next) - Next.js-spezifische Regeln
- [`ultracite/solid`](/preset/solid) - Solid.js-Regeln
- [`ultracite/vue`](/preset/vue) - Vue.js-Regeln
- [`ultracite/svelte`](/preset/svelte) - Svelte-Regeln
- [`ultracite/qwik`](/preset/qwik) - Qwik-Regeln
- [`ultracite/angular`](/preset/angular) - Angular HTML-Parser
- [`ultracite/remix`](/preset/remix) - Unterstützung für dateibasiertes Routing

## Migration

### Next.js/React-Projekte

Aktualisiere deine `biome.jsonc`, um die entsprechenden Presets zu erweitern:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core", "ultracite/react", "ultracite/next"]
}
```

### Nicht-React-Projekte

Wechsle zum Core-Preset, wenn du React nicht verwendest:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core"]
}
```

### Andere Frameworks

Verwende das passende Preset für dein Framework:

```jsonc title="biome.jsonc"
{
  // Solid.js
  "extends": ["ultracite/core", "ultracite/solid"]

  // Vue.js
  "extends": ["ultracite/core", "ultracite/vue"]

  // Svelte
  "extends": ["ultracite/core", "ultracite/svelte"]

  // Qwik
  "extends": ["ultracite/core", "ultracite/qwik"]

  // Angular
  "extends": ["ultracite/core", "ultracite/angular"]
}
```

### Multi-Framework-Projekte

Kombiniere Presets nach Bedarf:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core", "ultracite/react", "ultracite/solid"]
}
```
