Ultracite v6 uvodi [presete specifične za framework](/configuration#framework-specific-configurations) koje vam daju veću kontrolu nad tim koja pravila važe za vaš projekat. **Ovo ažuriranje ne prekida kompatibilnost** za korisnike Next.js/React.

## Šta se promenilo

v6 uvodi [presete specifične za framework](/configuration#framework-specific-configurations):

- [`ultracite/core`](/preset/core) - Osnovna pravila za JavaScript/TypeScript
- [`ultracite/react`](/preset/react) - Pravila specifična za React
- [`ultracite/next`](/preset/next) - Pravila specifična za Next.js
- [`ultracite/solid`](/preset/solid) - Pravila za Solid.js
- [`ultracite/vue`](/preset/vue) - Pravila za Vue.js
- [`ultracite/svelte`](/preset/svelte) - Pravila za Svelte
- [`ultracite/qwik`](/preset/qwik) - Pravila za Qwik
- [`ultracite/angular`](/preset/angular) - HTML parser za Angular
- [`ultracite/remix`](/preset/remix) - Podrška za rutiranje zasnovano na fajlovima

## Migracija

### Next.js/React projekti

Ažurirajte vaš `biome.jsonc` da proširite odgovarajuće presete:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core", "ultracite/react", "ultracite/next"]
}
```

### Projekti bez React-a

Pređite na osnovni preset ako ne koristite React:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core"]
}
```

### Ostali frameworki

Koristite odgovarajući preset za vaš framework:

```jsonc title="biome.jsonc"
{
  // Solid.js
  "extends": ["ultracite/core", "ultracite/solid"]

  // Vue.js
  "extends": ["ultracite/core", "ultracite/vue"]

  // Svelte
  "extends": ["ultracite/core", "ultracite/svelte"]

  // Qwik
  "extends": ["ultracite/core", "ultracite/qwik"]

  // Angular
  "extends": ["ultracite/core", "ultracite/angular"]
}
```

### Projekti sa više frameworka

Kombinujte presete po potrebi:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core", "ultracite/react", "ultracite/solid"]
}
```
