Az Angular konfiguráció Angular-specifikus HTML-elemző beállításokat tartalmaz az Angular-sablon szintaxisának támogatásához.

## Telepítés

Add hozzá az Angular konfigurációt a `biome.jsonc` fájlodhoz:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core", "ultracite/angular"]
}
```

## Áttekintés

Ez a konfiguráció támogatást nyújt az Angular sablonszintaxisához:

- **HTML interpoláció**: Lehetővé teszi az Angular sablon interpolációs szintaxisának (`{{ }}`) elemzését
- **Sablon direktívák**: Támogatja az olyan Angular direktívákat, mint `*ngIf`, `*ngFor` stb.
- **Tulajdonságkötés**: Kezeli az Angular tulajdonságkötési szintaxist `[property]="value"`
- **Eseménykötés**: Támogatja az eseménykötési szintaxist `(event)="handler()"`

## Konfiguráció

Az Angular előbeállítás beállítja a HTML-elemzőt, hogy felismerje az Angular sabloninterpolációt:

```jsonc
{
  "html": {
    "parser": {
      "interpolation": true
    }
  }
}
```

Ez lehetővé teszi a Biome számára, hogy helyesen elemezze és formázza az Angular komponenssablonokat anélkül, hogy az interpolációs szintaxist hibaként kezelné.
