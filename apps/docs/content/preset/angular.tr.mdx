---

Angular yapılandırması, Angular şablon sözdizimini desteklemek için Angular'a özgü HTML ayrıştırıcı ayarlarına sahiptir.

## Kurulum

Angular yapılandırmasını `biome.jsonc` dosyanıza ekleyin:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core", "ultracite/angular"]
}
```

## Genel Bakış

Bu yapılandırma Angular'ın şablon sözdizimini destekler:

- **HTML İnterpolasyonu**: Angular şablon interpolasyon sözdizimini (`{{ }}`) ayrıştırmayı etkinleştirir
- **Şablon Yönergeleri**: `*ngIf`, `*ngFor` gibi Angular yönergelerini destekler
- **Özellik Bağlama**: Angular özellik bağlama sözdizimini `[property]="value"` işler
- **Olay Bağlama**: `(event)="handler()"` gibi olay bağlama sözdizimini destekler

## Yapılandırma

Angular ön ayarı, HTML ayrıştırıcısını Angular'ın şablon interpolasyonunu tanıyacak şekilde yapılandırır:

```jsonc
{
  "html": {
    "parser": {
      "interpolation": true
    }
  }
}
```

Bu, Biome'nin interpolasyon sözdizimini hata olarak değerlendirmeden Angular bileşen şablonlarını düzgün şekilde ayrıştırıp biçimlendirmesini sağlar.
