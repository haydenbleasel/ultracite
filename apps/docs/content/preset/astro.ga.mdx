---

Tá coigeartuithe i rialacha linting shonracha Astro sa chumraíocht seo chun struchtúr uathúil chomhpháirteanna Astro a láimhseáil, go háirithe timpeall scripteanna frontmatter agus sintéis na gcomhpháirteanna.

## Suiteáil

Cuir an chumraíocht Astro leis do `biome.jsonc`:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core", "ultracite/astro"]
}
```

## Forbhreathnú

Déantar coigeartuithe ar rialacha don fhorbairt Astro sa chumraíocht seo:

- **Athróga agus allmhairí neamhúsáidte**: Ceadaítear athróga agus allmhairí neamhúsáidte i gcomhaid Astro de bharr srianta frontmatter ina bhféadfadh athróga a bheith in úsáid ach sa chuid teimpléid
- **Fógraí athróg**: Laghdaíonn sé an éilimh ar `const` ós rud é go bhféadfadh frontmatter Astro a bheith ag teastáil athróga a fhéadfar a athshannadh
- **Cineálacha allmhairithe**: Múchann sé éilimh shonracha ar chineál allmhairithe chun comhoiriúnacht níos fearr a chinntiú
- **Airíonna sonracha do React**: Tugann sé foláirimh faoi airíonna sonracha React cosúil le `className` ós rud é go n-úsáideann Astro tréithe HTML caighdeánacha

## Rialacha Sonracha Astro

### Ceartas

| Rule | Setting | Description |
|------|---------|-------------|
| `noUnusedVariables` | `off` | Ceadaigh athróga neamhúsáidte i gcomhaid Astro. D’fhéadfadh athróga a fógraíodh sa frontmatter a úsáid ach sa chuid teimpléid. |
| `noUnusedImports` | `off` | Ceadaigh allmhairí neamhúsáidte i gcomhaid Astro. D’fhéadfadh allmhairí sa frontmatter a úsáid ach sa chuid teimpléid. |

### Stíl

| Rule | Setting | Description |
|------|---------|-------------|
| `useConst` | `off` | Ceadaigh fógraí `let` agus `var`. D’fhéadfadh go mbeadh gá le athróga inathshanntaithe i frontmatter comhpháirte Astro. |
| `useImportType` | `off` | Múchann sé riachtanais shonracha maidir le cineál allmhairithe chun comhoiriúnacht níos fearr le córas tógála Astro a chinntiú. |

### Amhrasach

| Rule | Setting | Description |
|------|---------|-------------|
| `noReactSpecificProps` | `error` | Diúltaíonn sé airíonna sonracha React cosúil le `className` agus `htmlFor` in Astro. Úsáid `class` agus `for` ina áit. |
