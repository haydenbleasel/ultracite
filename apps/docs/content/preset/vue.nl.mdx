De Vue-configuratie bevat Vue.js-specifieke lintregels voor componentstructuur, data-opties, berekende eigenschappen en gereserveerde sleutelwoorden. Het bevat ook overrides om valse positieve lintfouten in `.vue`-bestanden te voorkomen.

## Installatie

Voeg de Vue-configuratie toe aan je `biome.jsonc`:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core", "ultracite/vue"]
}
```

## Overzicht

Deze configuratie voegt regels toe die specifiek zijn voor Vue.js-ontwikkeling:

- **HTML-ondersteuning**: Activeert experimentele volledige HTML-ondersteuning met inspringing voor script- en style-tags
- **React-specifieke props**: Verbiedt React-specifieke props in Vue-componenten
- **Overrides voor Vue-bestanden**: Versoepelde regels voor `.vue`-bestanden om de Single File Component-structuur af te handelen

## HTML-formatteringsconfiguratie

De Vue-preset schakelt experimentele HTML-ondersteuning in met speciale opmaak voor ingesloten scripts en styles:

```jsonc
{
  "html": {
    "experimentalFullSupportEnabled": true,
    "formatter": {
      "indentScriptAndStyle": true
    }
  }
}
```

Deze configuratie:
- **Experimentele volledige ondersteuning**: Schakelt de experimentele HTML-parsing- en formatteringsmogelijkheden van Biome in voor `.vue`-bestanden
- **Inspringing van script en style**: Zorgt voor correcte inspringing van code binnen `<script>`- en `<style>`-tags in Vue Single File Components

## Vue-specifieke regels

### Verdacht

| Regel | Instelling | Beschrijving |
|------|---------|-------------|
| `noReactSpecificProps` | `error` | Verbiedt React-specifieke props zoals `className` en `htmlFor` in Vue. Gebruik in plaats daarvan `class` en `for`. |

## Overrides voor Vue-bestanden

De volgende regels zijn uitgeschakeld voor `.vue`-bestanden om valse positieven te voorkomen vanwege Biome's gedeeltelijke ondersteuning voor Vue Single File Components:

### Correctheid

| Regel | Instelling | Beschrijving |
|------|---------|-------------|
| `noUnusedVariables` | `off` | Sta ongebruikte variabelen toe in `.vue`-bestanden. Variabelen in het scriptgedeelte kunnen alleen in de template worden gebruikt. |
| `noUnusedImports` | `off` | Sta ongebruikte imports toe in `.vue`-bestanden. Imports kunnen alleen in het templategedeelte worden gebruikt. |

### Stijl

| Regel | Instelling | Beschrijving |
|------|---------|-------------|
| `useConst` | `off` | Sta `let`- en `var`-declaraties toe in `.vue`-bestanden voor reactieve datapatronen. |
| `useImportType` | `off` | Schakel expliciete import-typevereisten uit voor betere compatibiliteit met het buildsysteem van Vue. |
