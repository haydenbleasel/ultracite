Tá sa chumraíocht React rialacha iniúchta sonracha do JSX, hooks, agus patrúin chomhpháirte.

## Suiteáil

Cuir an chumraíocht React le do `biome.jsonc`:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core", "ultracite/react"]
}
```

## Forbhreathnú

Cuir an chumraíocht seo rialacha sonracha leis don fhorbairt React:

- **Rialacha JSX**: Cur chun feidhme ar shintéis agus ar phatrúin JSX chuí
- **Rialacha Hooks**: Déan cinnte go n-úsáidtear hooks React i gceart
- **Rialacha Comhpháirte**: Cur chun feidhme ar na cleachtais is fearr do chomhpháirteanna React
- **Láimhseáil children**: Cosc ar úsáid mhíchuí an prop children

## Rialacha Sonracha React

### Ceartúlacht

| Rialach | Socrú | Cur síos |
|--------|--------|----------|
| `noChildrenProp` | `error` | Cosc ar tharchur children mar prop. Ba chóir go mbeadh children suiteáilte idir an lipéad oscailte agus an lipéad dúnach. |
| `noNestedComponentDefinitions` | `error` | Cosc ar shainmhínithe comhpháirt laistigh de chomhpháirteanna eile. Cruthaíonn sé seo ath-rindreálacha neamh-riachtanacha. |
| `noReactPropAssignments` | `error` | Cosc ar athcheapadh props i gcomhpháirteanna React. Ba chóir props a chóireáil mar neamh-athraitheach. |
| `noRenderReturnValue` | `error` | Cosc ar úsáid luach fillte ó `ReactDOM.render()`. |
| `noVoidElementsWithChildren` | `error` | Cosc ar eilimintí folamh (mar `<img>`, `<br>`) a bheith le children. |
| `useExhaustiveDependencies` | `error` | Cur chun feidhme go bhfuil na spleáchais go léir sonraithe i gceart i hooks React (useEffect, useCallback, useMemo). |
| `useHookAtTopLevel` | `error` | Cur chun feidhme gur gá glaoch ar gach hook React ón leibhéal barr de fheidhmeanna comhpháirte, ní laistigh de lúbanna, coinníollacha, nó feidhmeanna faoistruchtúrtha. |
| `useJsxKeyInIterable` | `error` | Cur chun feidhme go bhfuil prop `key` ag eilimintí i iterable chun comhfhreagras React a chumasú. |

### Stíl

| Rialach | Socrú | Cur síos |
|--------|--------|----------|
| `useReactFunctionComponents` | `error` | Cur chun feidhme úsáid a bhaint as comhpháirteanna feidhme in ionad comhpháirteanna rang i React. |

### Amhrasach

| Rialach | Socrú | Cur síos |
|--------|--------|----------|
| `noArrayIndexKey` | `error` | Cosc ar úsáid innéacsí eagar mar keys. Níl innéacsí eagar mar aitheantóirí seasta agus féadfaidh siad fadhbanna a chruthú le staid comhpháirte. |
| `noDuplicateJsxProps` | `error` | Cosc ar airíonna dúblacha i JSX. |
| `noSuspiciousSemicolonInJsx` | `error` | Cosc ar sheimicólóin a athraíonn brí eilimintí JSX. |
| `noReactSpecificProps` | `off` | Ceadaigh props sonracha React cosúil le `className` agus `htmlFor`. |
