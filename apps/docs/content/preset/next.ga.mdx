Tá sa chumraíocht Next.js rialacha lintála sonracha do bharrfheabhsú íomhá, struchtúr doiciméid, agus patrúin App Router.

## Suiteáil

Cuir an chumraíocht Next.js le do `biome.jsonc`:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core", "ultracite/next"]
}
```

## Forbhreathnú

Cuirfidh an chumraíocht seo rialacha sonracha ar fáil le haghaidh forbartha Next.js:

- **Barrfheabhsú Íomhá**: Éiligh úsáid an chomhlachta Image de Next.js le haghaidh barrfheabhsú íomhá uathoibríoch
- **Struchtúr Doiciméid**: Cosc ar úsáid míchuí de chomhlachtaí speisialta Next.js
- **Comhaid Cumraíochta**: Láimhseáil speisialta do chomhaid `next.config.*`

## Rialacha Sonracha Next.js

### Feidhmíocht

| Rule | Setting | Description |
|------|---------|-------------|
| `noImgElement` | `error` | Ní cheadaítear úsáid an eilimint HTML `<img>`. Úsáid an chomhlacht `<Image>` de Next.js in áit sin le haghaidh barrfheabhsú íomhá uathoibríoch. |

### Stíl

| Rule | Setting | Description |
|------|---------|-------------|
| `noHeadElement` | `error` | Ní cheadaítear úsáid an eilimint HTML `<head>`. Úsáid `next/head` de Next.js nó API metadata App Router ina áit. |

### Amhrasach

| Rule | Setting | Description |
|------|---------|-------------|
| `noDocumentImportInPage` | `error` | Cosc ar allmhairiú `next/document` i gcomhaid leathanach. Ba chóir go mbeadh `_document.tsx` ann ach amháin i `pages/_document.tsx`. |
| `noHeadImportInDocument` | `error` | Cosc ar allmhairiú `next/head` i `_document.tsx`. Úsáid `<Head>` ó `next/document` ina áit. |

## Sárshocruithe Comhad Cumraíochta

Cuimsíonn an réamhshocrú Next.js láimhseáil speisialta do chomhaid chumraíochta agus do chomhlachtaí App Router:

### Comhaid Chumraíochta Next.js

```jsonc
{
  "overrides": [
    {
      "includes": ["next.config.*"],
      "linter": {
        "rules": {
          "suspicious": {
            "useAwait": "off"
          }
        }
      }
    }
  ]
}
```

Ligeann sé seo patrúin chumraíochta Next.js nach bhféadfadh de ghnáth pas a fháil trí rialacha lintála.

### Leathanaigh agus Leaganacha App Router

```jsonc
{
  "overrides": [
    {
      "includes": ["**/page.{ts,tsx,js,jsx}", "**/layout.{ts,tsx,js,jsx}"],
      "linter": {
        "rules": {
          "suspicious": {
            "useAwait": "off"
          }
        }
      }
    }
  ]
}
```

Ceadaítear go mbeidh comhlachtaí leathanach agus leaganacha App Router async gan gá le `await`, toisc go láimhseálann Next.js iad seo go speisialta le haghaidh rindreáil taobh-fhreastalaí agus fáil sonraí.
