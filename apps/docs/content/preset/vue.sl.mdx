Konfiguracija za Vue vsebuje pravila lintanja, specifična za Vue.js za strukturo komponent, možnosti podatkov, izpeljane lastnosti in rezervirane ključne besede. Vključuje tudi preglasitve za preprečevanje lažno pozitivnih napak pri lintanju v `.vue` datotekah.

## Namestitev

Dodajte Vue konfiguracijo v vašo `biome.jsonc`:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core", "ultracite/vue"]
}
```

## Pregled

Ta konfiguracija doda pravila, specifična za razvoj z Vue.js:

- **Podpora za HTML**: Omogoči eksperimentalno polno podporo za HTML z zamikom za script in style
- **React-specifične props**: Onemogoči React-specifične props v Vue komponentah
- **Preglasitve za Vue datoteke**: Omiljena pravila za `.vue` datoteke, da se obravnava struktura Single File Component

## Konfiguracija oblikovanja HTML

Prednastavitev za Vue omogoča eksperimentalno podporo za HTML s posebno obliko za vdelane skripte in stile:

```jsonc
{
  "html": {
    "experimentalFullSupportEnabled": true,
    "formatter": {
      "indentScriptAndStyle": true
    }
  }
}
```

Ta konfiguracija:
- **Eksperimentalna polna podpora**: Omogoči eksperimentalne zmožnosti Biome za razčlenjevanje in oblikovanje HTML za `.vue` datoteke
- **Zamik za script in style**: Pravilen zamik kode znotraj `<script>` in `<style>` tagov v Vue Single File Componentih

## Pravila specifična za Vue

### Sumljivo

| Pravilo | Nastavitev | Opis |
|------|---------|-------------|
| `noReactSpecificProps` | `error` | Prepoveduje React-specifične props, kot so `className` in `htmlFor`, v Vue. Uporabite `class` in `for` namesto tega. |

## Preglasitve za Vue datoteke

Naslednja pravila so onemogočena za `.vue` datoteke, da se preprečijo lažno pozitivne ugotovitve zaradi delne podpore Biome za Vue Single File Components:

### Korektnost

| Pravilo | Nastavitev | Opis |
|------|---------|-------------|
| `noUnusedVariables` | `off` | Dovoli neuporabljene spremenljivke v `.vue` datotekah. Spremenljivke v razdelku script se lahko uporabljajo samo v predlogi (template). |
| `noUnusedImports` | `off` | Dovoli neuporabljene uvoze v `.vue` datotekah. Uvozi se lahko uporabljajo samo v razdelku s predlogo (template). |

### Slog

| Pravilo | Nastavitev | Opis |
|------|---------|-------------|
| `useConst` | `off` | Dovoli deklaracije `let` in `var` v `.vue` datotekah za vzorce reaktivnih podatkov. |
| `useImportType` | `off` | Onemogoči zahteve po eksplicitnih tipih uvoza za boljšo združljivost z Vue sistemom za gradnjo. |
