---

Soláthraíonn cumraíocht chroí Ultracite sraith chuimsitheach rialacha lintála neamhspleácha ar fhrámaí agus socruithe formáide, agus is í seo an bonn a leathnaíonn gach cumraíocht shonrach do fhráma eile.

## Suiteáil

Tá an chumraíocht chroí san áireamh go huathoibríoch nuair a rith tú:

```bash
npx ultracite init
```

Nó cuir leis de láimh chuig do `biome.jsonc`:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core"]
}
```

## Forbhreathnú

Cuimsíonn an chumraíocht chroí na céadta rialacha lintála dian a chlúdaíonn:

- **Inrochtaineacht (a11y)**: Cuir i bhfeidhm airíonna ARIA, HTML seiméantach, agus nascleanúint méarchláir
- **Castaíocht**: Laghdaigh castaíocht chognaíoch agus cuir i bhfeidhm cleachtais is fearr
- **Cruinneas**: Cosain i gcoinne earráidí coitianta agus cuir i bhfeidhm sábháilteacht chineál
- **Feidhmíocht**: Optamóidh cód le haghaidh feidhmíocht ama rith níos fearr
- **Slándáil**: Cosain i gcoinne laigí slándála
- **Stíl**: Cuir i bhfeidhm stíl chód comhsheasmhach
- **Amhrasach**: Gabh patrúin amhrasacha a d’fhéadfadh earráidí a léiriú

## Socruithe Formáidóra

- **Indentation**: 2 spás
- **Line Width**: 80 carachtar
- **Line Ending**: LF (stíl Unix)
- **Semicolons**: I gcónaí riachtanach
- **Quotes**: Luachanna dúbailte do JavaScript/TypeScript agus JSON
- **Trailing Commas**: Stíl ES5 (suirbhéanna/objáidí illíne)
- **Arrow Parentheses**: I gcónaí cuir lúibíní san áireamh

## Príomhghnéithe

### Dáincheartas TypeScript

Cumasaíonn an chumraíocht chroí seiceálacha stríoca TypeScript:

- Spreagann sé neamhúsáid `any` (`noExplicitAny: error`)
- Éilíonn sé láimhseáil `null/undefined`
- Is fearr cineálacha sainráite i gcásanna áirithe
- Cuirtear sábháilteacht chineál i bhfeidhm ar fud an bhonnchód

### Comhionanna Molta ESLint

Tá rialacha dea-chleachtais choitianta san áireamh:

- Gan athróga neamhúsáide (`noUnusedVariables: error`)
- Gan ionchur neamhúsáidte (`noUnusedImports: error`)
- Gan eval shoiléir (`noGlobalEval: error`)
- Gan truailliú protoitéipe (`noPrototypeBuiltins: error`)

### Inrochtaineacht mar Réamhshocrú

Rialacha comhionanna do chaighdeáin inrochtaineachta:

- Ní mór airíonna ARIA a bheith bailí
- Is fearr eilimintí HTML seiméantacha
- Éilítear tacaíocht do nascleanúint méarchláir
- Éilítear téacs alt do íomhánna

### Tacaíocht Node.js

Láimhseáil speisialta do thimpeallacht:

- Aithníonn sé imlíneacha domhanda Node.js (`module`, `process`)
- Cuirtear an prótacal `node:` i bhfeidhm do mhodúil ionsuite
- Tacaíonn sé do CommonJS agus ESM araon

| `useIterableCallbackReturn` | `error` | Cuir i bhfeidhm luachanna tuairisce comhsheasmhacha i callbacks inIterable. |
| `useNamespaceKeyword` | `error` | Éilíonn sé úsáid an eochairfhocail namespace thar `module` chun ainmspásanna TypeScript a shonrú. |
| `useNumberToFixedDigitsArgument` | `error` | Éilíonn sé úsáid an argóint digits le `Number#toFixed()`. |
| `useStaticResponseMethods` | `error` | Úsáid modhanna statacha `Response` in ionad tógálaí `new Response()` nuair is féidir. |
| `useStrictMode` | `error` | Cuir i bhfeidhm an treoir `use strict` i gcomhaid script. |

## Nótaí

Tá roinnt rialacha díghníomhachtaithe mar tá siad ró-mhall le rith leis an Scanner. Mar atá luaite sna [docaí Biome](https://biomejs.dev/guides/investigate-slowness/#first-things-first), tá a fhios go mbíonn ormchumhacht feidhmíochta ag rialacha tionscadail a mheastar a bheith cumasaithe i mhalairt ar anailís níos forbartha.

I bhfrithghníomhú tástála, fuaireamar amach go bhféadfadh na rialacha seo an difríocht a dhéanamh idir 300ms agus 13s le linn lintála.
