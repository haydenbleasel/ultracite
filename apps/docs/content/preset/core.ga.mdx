Soláthraíonn cumraíocht chroí Ultracite tacar cuimsitheach rialacha lintála neamhchlaonta-iosctha agus socruithe formáidithe. Is é seo an bunús a leathnaíonn gach cumrú sonraíoch freagrachta eile.

## Suiteáil

Tá an chumraíocht chroí san áireamh go huathoibríoch nuair a ritheann tú:

```bash
npx ultracite init
```

Nó cuir leis de láimh i do `biome.jsonc`:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core"]
}
```

## Forbhreathnú

Cuimsíonn an chumraíocht chroí na céadta rialacha dian lintála a dhéanaíonn clúdach ar:

- **Inrochtaineacht (a11y)**: Forchuireann tréithe ARIA, HTML seimeantach, agus nascleanúint méarchláir
- **Castacht**: Laghdú ar chasta intleachtúil agus forchuimhnigh dea-chleachtais
- **Céartas**: Cosc ar earráidí coitianta agus forchuimhnigh sábháilteacht cineál
- **Feidhmíocht**: Optamú cód le haghaidh feidhmíochta rith níos fearr
- **Slándáil**: Cosc ar leochaileachtaí slándála
- **Stíl**: Forchuimhnigh stíl chód chomhsheasmhach
- **Amhrasach**: Gabhaimid patrúin amhrasacha a d’fhéadfadh a bheith ina fabhtanna

## Socruithe Formáidithe

- **Tabhairt isteach**: 2 spásanna
- **Leithead Líne**: 80 carachtar
- **Deiridh Líne**: LF (stíl Unix)
- **Seimicolúin**: Riachtanach i gcónaí
- **Luacháin**: Luacháin dhúbailte do JavaScript/TypeScript agus JSON
- **Camáin Deiridh**: Stíl ES5 (eagar/obair illíne)
- **Lúibíní Saigheada**: Cuir i gcónaí lúibíní san áireamh

## Príomhghnéithe

### Dianas TypeScript

Ceadaíonn an chumraíocht chroí seiceálacha dian TypeScript:

- Spreagann sé neamhúsáid `any` (`noExplicitAny: error`)
- Éilíonn sé láimhseáil `null/undefined`
- Cothaíonn sé cineálacha sainmhínithe i gcásanna áirithe
- Forchuireann sé sábháilteacht cineál ar fud an bhunachar cód

### Cothromóidí molta ESLint

Tá rialacha cleachtais is fearr coitianta san áireamh:

- Gan athróga neamhúsáidte (`noUnusedVariables: error`)
- Gan iompórtálacha neamhúsáidte (`noUnusedImports: error`)
- Gan eval shoiléir (`noGlobalEval: error`)
- Gan truailliú protacalóige (`noPrototypeBuiltins: error`)

### Inrochtaineacht de réir réamhshocraithe

Rialacha comhréireacha le caighdeáin inrochtaineachta:

- Ní mór go mbeadh tréithe ARIA bailí
- Tosaítear ar eilimintí HTML seimeantacha
- Éilítear tacaíocht nascleanúna méarchláir
- Éilítear téacs alt do íomhánna

### Tacaíocht Node.js

Láimhseáil shonrach don timpeallacht:

- Aithníonn sé glóibálacha Node.js (`module`, `process`)
- Forchuireann sé an prótacal `node:` do mhodúil ionsuite
- Tacaíonn sé le CommonJS agus ESM araon
