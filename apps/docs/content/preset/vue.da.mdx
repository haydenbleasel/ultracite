Vue-konfigurationen indeholder Vue.js-specifikke lintregler for komponentstruktur, dataindstillinger, beregnede egenskaber og reserverede nøgleord. Den inkluderer også overskrivelser for at forhindre falske positive lintfejl i `.vue`-filer.

## Installation

Tilføj Vue-konfigurationen til din `biome.jsonc`:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core", "ultracite/vue"]
}
```

## Oversigt

Denne konfiguration tilføjer regler, der er specifikke for Vue.js-udvikling:

- **HTML-understøttelse**: Aktiverer eksperimentel fuld HTML-understøttelse med indrykning af script- og style-sektioner
- **React-specifikke props**: Forbyd React-specifikke props i Vue-komponenter
- **Overskrivelser for Vue-filer**: Lemper regler for `.vue`-filer for at håndtere Single File Component-strukturen

## HTML-formatkonfiguration

Vue-preset'et aktiverer eksperimentel HTML-understøttelse med særlig formatering for indlejrede scripts og styles:

```jsonc
{
  "html": {
    "experimentalFullSupportEnabled": true,
    "formatter": {
      "indentScriptAndStyle": true
    }
  }
}
```

Denne konfiguration:
- **Eksperimentel fuld understøttelse**: Aktiverer Biome's eksperimentelle HTML-parsing og formatteringsmuligheder for `.vue`-filer
- **Indryk script og style**: Indrykker korrekt kode inde i `<script>`- og `<style>`-tags i Vue Single File Components

## Vue-specifikke regler

### Mistænkelige

| Regel | Indstilling | Beskrivelse |
|------|---------|-------------|
| `noReactSpecificProps` | `error` | Forbyd React-specifikke props som `className` og `htmlFor` i Vue. Brug `class` og `for` i stedet. |

## Overskrivelser for Vue-filer

Følgende regler er deaktiveret for `.vue`-filer for at forhindre falske positiver på grund af Biome's delvise understøttelse af Vue Single File-komponenter:

### Korrekthed

| Regel | Indstilling | Beskrivelse |
|------|---------|-------------|
| `noUnusedVariables` | `off` | Tillad ubrugte variabler i `.vue`-filer. Variabler i scriptsektionen kan kun bruges i template-sektionen. |
| `noUnusedImports` | `off` | Tillad ubrugte imports i `.vue`-filer. Imports kan kun bruges i template-sektionen. |

### Stil

| Regel | Indstilling | Beskrivelse |
|------|---------|-------------|
| `useConst` | `off` | Tillad `let` og `var`-deklarationer i `.vue`-filer til reaktive datamønstre. |
| `useImportType` | `off` | Deaktiver eksplicitte import-typekrav for bedre kompatibilitet med Vues buildsystem. |
