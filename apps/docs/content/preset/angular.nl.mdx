De Angular-configuratie bevat Angular-specifieke HTML-parserinstellingen om de Angular-templatesyntaxis te ondersteunen.

## Installatie

Voeg de Angular-configuratie toe aan uw `biome.jsonc`:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core", "ultracite/angular"]
}
```

## Overzicht

Deze configuratie voegt ondersteuning toe voor de templatesyntaxis van Angular:

- **HTML-interpolatie**: Schakelt het parsen in van de Angular-templateinterpolatiesyntaxis (`{{ }}`)
- **Template-directieven**: Ondersteunt Angular-directieven zoals `*ngIf`, `*ngFor`, enz.
- **Property-binding**: Verwerkt de Angular-property-binding-syntaxis `[property]="value"`
- **Event-binding**: Ondersteunt de event-binding-syntaxis `(event)="handler()"`

## Configuratie

De Angular-preset configureert de HTML-parser om de template-interpolatie van Angular te herkennen:

```jsonc
{
  "html": {
    "parser": {
      "interpolation": true
    }
  }
}
```

Dit stelt Biome in staat Angular-componenttemplates correct te parsen en op te maken zonder de interpolatiesyntaxis als fouten te behandelen.
