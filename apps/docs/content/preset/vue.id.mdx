Konfigurasi Vue memiliki aturan linting khusus Vue.js untuk struktur komponen, opsi data, properti computed, dan kata kunci yang dicadangkan. Konfigurasi ini juga menyertakan override untuk mencegah kesalahan linting positif palsu di file `.vue`.

## Instalasi

Tambahkan konfigurasi Vue ke `biome.jsonc` Anda:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core", "ultracite/vue"]
}
```

## Ikhtisar

Konfigurasi ini menambahkan aturan yang spesifik untuk pengembangan Vue.js:

- **Dukungan HTML**: Mengaktifkan dukungan HTML penuh eksperimental dengan indentasi pada script dan style
- **Properti Khusus React**: Melarang properti khusus React pada komponen Vue
- **Override File Vue**: Aturan dilonggarkan untuk file `.vue` agar menangani struktur Single File Component

## Konfigurasi Pemformat HTML

Preset Vue mengaktifkan dukungan HTML eksperimental dengan pemformatan khusus untuk script dan style yang disematkan:

```jsonc
{
  "html": {
    "experimentalFullSupportEnabled": true,
    "formatter": {
      "indentScriptAndStyle": true
    }
  }
}
```

Konfigurasi ini:
- **Dukungan Penuh Eksperimental**: Mengaktifkan kemampuan parsing dan pemformatan HTML eksperimental Biome untuk file `.vue`
- **Indentasi Script dan Style**: Mengindentasi kode di dalam tag `<script>` dan `<style>` dengan benar dalam Vue Single File Components

## Aturan Khusus Vue

### Mencurigakan

| Aturan | Pengaturan | Deskripsi |
|--------|-----------|-----------|
| `noReactSpecificProps` | `error` | Melarang properti khusus React seperti `className` dan `htmlFor` di Vue. Gunakan `class` dan `for` sebagai gantinya. |

## Penyesuaian untuk File Vue

Aturan berikut dinonaktifkan untuk file `.vue` untuk mencegah positif palsu karena dukungan parsial Biome untuk Vue Single File Components:

### Kebenaran

| Aturan | Pengaturan | Deskripsi |
|--------|-----------|-----------|
| `noUnusedVariables` | `off` | Mengizinkan variabel yang tidak terpakai di file `.vue`. Variabel di bagian script mungkin hanya digunakan di dalam template. |
| `noUnusedImports` | `off` | Mengizinkan impor yang tidak terpakai di file `.vue`. Impor mungkin hanya digunakan di bagian template. |

### Gaya

| Aturan | Pengaturan | Deskripsi |
|--------|-----------|-----------|
| `useConst` | `off` | Mengizinkan deklarasi `let` dan `var` di file `.vue` untuk pola data reaktif. |
| `useImportType` | `off` | Menonaktifkan persyaratan tipe impor eksplisit untuk kompatibilitas yang lebih baik dengan sistem build Vue. |
