Angular-konfigurasjonen har Angular-spesifikke HTML-parserinnstillinger for å støtte Angular-malsyntaks.

## Installasjon

Legg til Angular-konfigurasjonen i din `biome.jsonc`:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core", "ultracite/angular"]
}
```

## Oversikt

Denne konfigurasjonen legger til støtte for Angulars malsyntaks:

- **HTML-interpolasjon**: Aktiverer parsing av Angular-malinterpolasjonssyntaks (`{{ }}`)
- **Maledirektiver**: Støtter Angular-direktiver som `*ngIf`, `*ngFor`, osv.
- **Egenskapsbinding**: Håndterer Angulars syntaks for egenskapsbinding `[property]="value"`
- **Hendelsesbinding**: Støtter syntaks for hendelsesbinding `(event)="handler()"`

## Konfigurasjon

Angular-presetet konfigurerer HTML-parseren til å gjenkjenne Angulars malinterpolasjon:

```jsonc
{
  "html": {
    "parser": {
      "interpolation": true
    }
  }
}
```

Dette gjør at Biome kan korrekt tolke og formatere Angular-komponentmaler uten å behandle interpolasjonssyntaksen som feil.
