A React konfiguráció React-specifikus lintelési szabályokat tartalmaz JSX-hez, hookokhoz és komponensmintákhoz.

## Telepítés

Adja hozzá a React konfigurációt a `biome.jsonc` fájlhoz:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core", "ultracite/react"]
}
```

## Áttekintés

Ez a konfiguráció React-fejlesztéshez specifikus szabályokat ad hozzá:

- **JSX szabályok**: Biztosítja a megfelelő JSX-szintaxist és mintákat
- **Hookok szabályai**: Biztosítja, hogy a React hookokat helyesen használják
- **Komponens szabályok**: Érvényesíti a React komponensekre vonatkozó legjobb gyakorlatokat
- **Children kezelése**: Megakadályozza a children prop nem megfelelő használatát

## React-specifikus szabályok

### Helyesség

| Szabály | Beállítás | Leírás |
|--------|-----------|--------|
| `noChildrenProp` | `error` | Megakadályozza, hogy a children propként legyen átadva. A children-nek a nyitó és záró tagek közé kell kerülnie. |
| `noNestedComponentDefinitions` | `error` | Megakadályozza, hogy komponensdefiníciók más komponensek belsejében legyenek. Ez felesleges újrarendereléseket okoz. |
| `noReactPropAssignments` | `error` | Megakadályozza a props újrahozzárendelését React komponensekben. A props-okat nem módosíthatóként kell kezelni. |
| `noRenderReturnValue` | `error` | Megakadályozza a `ReactDOM.render()` visszatérési értékének használatát. |
| `noVoidElementsWithChildren` | `error` | Megakadályozza, hogy void elemek (például `<img>`, `<br>`) children-t tartalmazzanak. |
| `useExhaustiveDependencies` | `error` | Megköveteli, hogy minden függőség helyesen legyen megadva a React hookokban (useEffect, useCallback, useMemo). |
| `useHookAtTopLevel` | `error` | Megköveteli, hogy minden React hookot a komponensfüggvények felső szintjén hívjanak meg, ne ciklusok, feltételek vagy beágyazott függvények belsejében. |
| `useJsxKeyInIterable` | `error` | Megköveteli, hogy az iterálható elemek rendelkezzenek `key` propbal a React összeegyeztetéséhez. |

### Stílus

| Szabály | Beállítás | Leírás |
|--------|-----------|--------|
| `useReactFunctionComponents` | `error` | Megköveteli a függvénykomponensek használatát az osztálykomponensek helyett a React-ben. |

### Gyanús

| Szabály | Beállítás | Leírás |
|--------|-----------|--------|
| `noArrayIndexKey` | `error` | Megakadályozza, hogy tömbindexeket használjanak kulcsként. A tömbindexek nem stabil azonosítók és problémákat okozhatnak a komponens állapotával. |
| `noDuplicateJsxProps` | `error` | Megakadályozza a duplikált tulajdonságokat JSX-ben. |
| `noSuspiciousSemicolonInJsx` | `error` | Megakadályozza a pontosvesszőket, amelyek megváltoztatják a JSX elemek szemantikáját. |
| `noReactSpecificProps` | `off` | Engedélyezi a React-specifikus propokat, mint a `className` és `htmlFor`. |
