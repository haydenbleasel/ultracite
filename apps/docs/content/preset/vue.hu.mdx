A Vue konfiguráció Vue.js-specifikus lintelési szabályokat tartalmaz a komponensszerkezet, adatopciók, computed tulajdonságok és fenntartott kulcsszavak számára. Tartalmaz továbbá felülírásokat is, hogy megakadályozza a hamis pozitív lintelési hibákat a `.vue` fájlokban.

## Telepítés

Adja hozzá a Vue konfigurációt a `biome.jsonc` fájlhoz:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core", "ultracite/vue"]
}
```

## Áttekintés

Ez a konfiguráció Vue.js fejlesztésre specifikus szabályokat ad hozzá:

- **HTML Support**: Engedélyezi a kísérleti teljes HTML-támogatást script és style behúzással
- **React-Specific Props**: Megtiltja a React-specifikus propok használatát Vue komponensekben
- **Vue File Overrides**: Enyhített szabályok a `.vue` fájlokhoz a Single File Component szerkezet kezelésére

## HTML formázó konfiguráció

A Vue preset engedélyezi a kísérleti HTML-támogatást beágyazott script- és style-részek különleges formázásával:

```jsonc
{
  "html": {
    "experimentalFullSupportEnabled": true,
    "formatter": {
      "indentScriptAndStyle": true
    }
  }
}
```

Ez a konfiguráció:
- **Experimental Full Support**: Engedélyezi a Biome kísérleti HTML elemzési és formázási képességeit a `.vue` fájlokhoz
- **Indent Script and Style**: Megfelelően behúzza a `<script>` és `<style>` tagek belsejében lévő kódot Vue Single File Componentekben

## Vue-specifikus szabályok

### Gyanús

| Szabály | Beállítás | Leírás |
|--------:|:---------:|:-------|
| `noReactSpecificProps` | `error` | Megtiltja olyan React-specifikus propokat, mint a `className` és `htmlFor` Vue-ban. Használja helyettük a `class` és `for`-t. |

## Vue fájlokra vonatkozó felülírások

A következő szabályok ki vannak kapcsolva a `.vue` fájlokra, hogy megakadályozzuk a hamis pozitív lintelési hibákat, amelyeket a Biome Vue Single File Componentek részleges támogatása okozhat:

### Helyesség

| Szabály | Beállítás | Leírás |
|--------:|:---------:|:-------|
| `noUnusedVariables` | `off` | Engedélyezi a nem használt változókat `.vue` fájlokban. A script szekcióban lévő változók előfordulhat, hogy csak a template-ben vannak használva. |
| `noUnusedImports` | `off` | Engedélyezi a nem használt importokat `.vue` fájlokban. Az importok előfordulhat, hogy csak a template szekcióban vannak használva. |

### Stílus

| Szabály | Beállítás | Leírás |
|--------:|:---------:|:-------|
| `useConst` | `off` | Engedélyezi a `let` és `var` deklarációkat `.vue` fájlokban a reaktív adatmintákhoz. |
| `useImportType` | `off` | Letiltja az explicit import típuskövetelményeket a jobb kompatibilitás érdekében a Vue build rendszerével. |
