React konfiguracija sadrži pravila lintera specifična za React za JSX, hook-ove i obrasce komponenti.

## Instalacija

Dodajte React konfiguraciju u vaš `biome.jsonc`:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core", "ultracite/react"]
}
```

## Pregled

Ova konfiguracija dodaje pravila specifična za React razvoj:

- **JSX pravila**: Obezbeđuju pravilnu JSX sintaksu i obrasce
- **Pravila za Hooks**: Obezbeđuju pravilnu upotrebu React hook-ova
- **Pravila komponenti**: Obezbeđuju najbolje prakse za React komponente
- **Rukovanje children**: Sprečava nepravilnu upotrebu children prop

## Pravila specifična za React

### Ispravnost

| Pravilo | Podešavanje | Opis |
|------|---------|-------------|
| `noChildrenProp` | `error` | Sprečava prosleđivanje children kao props. Children treba da budu ugnježdeni između otvarajućeg i zatvarajućeg taga. |
| `noNestedComponentDefinitions` | `error` | Sprečava definisanje komponenti unutar drugih komponenti. To dovodi do nepotrebnih ponovnih renderovanja. |
| `noReactPropAssignments` | `error` | Sprečava ponovno dodeljivanje props u React komponentama. Props treba tretirati kao nepromenljive. |
| `noRenderReturnValue` | `error` | Sprečava upotrebu vrednosti koju vraća `ReactDOM.render()`. |
| `noVoidElementsWithChildren` | `error` | Sprečava da void elementi (kao `<img>`, `<br>`) imaju children. |
| `useExhaustiveDependencies` | `error` | Zahteva da sve zavisnosti budu ispravno navedene u React hook-ovima (useEffect, useCallback, useMemo). |
| `useHookAtTopLevel` | `error` | Zahteva da se svi React hook-ovi pozivaju na vrhunskom nivou funkcija komponenti, ne unutar petlji, uslova ili ugnježdenih funkcija. |
| `useJsxKeyInIterable` | `error` | Zahteva da elementi u iterabilima imaju `key` prop radi Reactove rekonsilijacije. |

### Stil

| Pravilo | Podešavanje | Opis |
|------|---------|-------------|
| `useReactFunctionComponents` | `error` | Zahteva korišćenje funkcionalnih komponenti umesto klasnih komponenti u Reactu. |

### Sumnjivo

| Pravilo | Podešavanje | Opis |
|------|---------|-------------|
| `noArrayIndexKey` | `error` | Sprečava korišćenje indeksa niza kao key vrednosti. Indeksi niza nisu stabilni identifikatori i mogu izazvati probleme sa stanjem komponenti. |
| `noDuplicateJsxProps` | `error` | Sprečava duplikate svojstava u JSX-u. |
| `noSuspiciousSemicolonInJsx` | `error` | Sprečava tačke-zareze koje menjaju semantiku JSX elemenata. |
| `noReactSpecificProps` | `off` | Dozvoljava React-specifične props kao što su `className` i `htmlFor`. |
