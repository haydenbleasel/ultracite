La configuración de Angular incluye ajustes específicos del analizador HTML para soportar la sintaxis de plantillas de Angular.

## Instalación

Añade la configuración de Angular a tu `biome.jsonc`:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core", "ultracite/angular"]
}
```

## Descripción general

Esta configuración añade soporte para la sintaxis de plantillas de Angular:

- **Interpolación HTML**: Habilita el análisis de la sintaxis de interpolación de plantillas de Angular (`{{ }}`)
- **Directivas de plantilla**: Admite directivas de Angular como `*ngIf`, `*ngFor`, etc.
- **Vinculación de propiedades**: Gestiona la sintaxis de vinculación de propiedades de Angular `[property]="value"`
- **Vinculación de eventos**: Admite la sintaxis de vinculación de eventos `(event)="handler()"`

## Configuración

El preset de Angular configura el analizador HTML para reconocer la interpolación de plantillas de Angular:

```jsonc
{
  "html": {
    "parser": {
      "interpolation": true
    }
  }
}
```

Esto permite a Biome analizar y formatear correctamente las plantillas de los componentes de Angular sin tratar la sintaxis de interpolación como errores.
