Next.js konfiguracija sadrži Next.js-specifična pravila lintovanja za optimizaciju slika, strukturu dokumenta i obrasce App Router-a.

## Instalacija

Dodajte Next.js konfiguraciju u vaš `biome.jsonc`:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core", "ultracite/next"]
}
```

## Pregled

Ova konfiguracija dodaje pravila specifična za Next.js razvoj:

- **Optimizacija slika**: Zahteva upotrebu Next.js Image komponente
- **Struktura dokumenta**: Sprečava nepravilnu upotrebu Next.js specijalnih komponenti
- **Konfiguracione datoteke**: Posebno rukovanje za `next.config.*` datoteke

## Pravila specifična za Next.js

### Performanse

| Pravilo | Podešavanje | Opis |
|---------|------------|------|
| `noImgElement` | `error` | Zabrani upotrebu `<img>` HTML elementa. Umesto toga koristite Next.js `<Image>` komponentu radi automatske optimizacije slika. |

### Stil

| Pravilo | Podešavanje | Opis |
|---------|------------|------|
| `noHeadElement` | `error` | Zabrani upotrebu `<head>` HTML elementa. Umesto toga koristite `next/head` ili App Router metadata API. |

### Sumnjivo

| Pravilo | Podešavanje | Opis |
|---------|------------|------|
| `noDocumentImportInPage` | `error` | Sprečava uvoz `next/document` u fajlove stranica. `_document.tsx` bi trebalo da postoji samo u `pages/_document.tsx`. |
| `noHeadImportInDocument` | `error` | Sprečava uvoz `next/head` u `_document.tsx`. Umesto toga koristite `<Head>` iz `next/document`. |

## Preklapanja konfiguracionih datoteka

Preset za Next.js uključuje posebno rukovanje za konfiguracione datoteke i App Router komponente:

### Next.js konfiguracione datoteke

```jsonc
{
  "overrides": [
    {
      "includes": ["next.config.*"],
      "linter": {
        "rules": {
          "suspicious": {
            "useAwait": "off"
          }
        }
      }
    }
  ]
}
```

Ovo dozvoljava Next.js obrasce konfiguracije koji obično možda ne bi prošli pravila lintovanja.

### App Router stranice i layouti

```jsonc
{
  "overrides": [
    {
      "includes": ["**/page.{ts,tsx,js,jsx}", "**/layout.{ts,tsx,js,jsx}"],
      "linter": {
        "rules": {
          "suspicious": {
            "useAwait": "off"
          }
        }
      }
    }
  ]
}
```

Komponente stranica i layout-a App Router-a su dozvoljene da budu async bez potrebe za `await`, jer Next.js posebno obrađuje ove slučajeve za server-side rendering i dohvat podataka.
