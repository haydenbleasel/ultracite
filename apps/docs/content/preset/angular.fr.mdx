La configuration Angular comporte des paramètres spécifiques du parseur HTML pour prendre en charge la syntaxe des templates Angular.

## Installation

Ajoutez la configuration Angular à votre `biome.jsonc`:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core", "ultracite/angular"]
}
```

## Aperçu

Cette configuration ajoute la prise en charge de la syntaxe des templates Angular :

- **Interpolation HTML**: Permet l'analyse de la syntaxe d'interpolation des templates Angular (`{{ }}`)
- **Directives de template**: Prend en charge les directives Angular comme `*ngIf`, `*ngFor`, etc.
- **Liaison de propriétés**: Gère la syntaxe de liaison de propriétés Angular `[property]="value"`
- **Liaison d'événements**: Prend en charge la syntaxe de liaison d'événements `(event)="handler()"`

## Configuration

Le préréglage Angular configure le parseur HTML pour reconnaître l'interpolation des templates Angular :

```jsonc
{
  "html": {
    "parser": {
      "interpolation": true
    }
  }
}
```

Cela permet à Biome d'analyser et de formater correctement les templates de composants Angular sans considérer la syntaxe d'interpolation comme des erreurs.
