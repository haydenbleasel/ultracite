Angular 配置具有特定于 Angular 的 HTML 解析器设置，以支持 Angular 模板语法。

## 安装

将 Angular 配置添加到您的 `biome.jsonc`：

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core", "ultracite/angular"]
}
```

## 概览

此配置为 Angular 的模板语法添加支持：

- **HTML 插值**: 启用对 Angular 模板插值语法 (`{{ }}`) 的解析
- **模板指令**: 支持像 `*ngIf`、`*ngFor` 等 Angular 指令
- **属性绑定**: 处理 Angular 的属性绑定语法 `[property]="value"`
- **事件绑定**: 支持事件绑定语法 `(event)="handler()"`

## 配置

Angular 预设将 HTML 解析器配置为识别 Angular 的模板插值：

```jsonc
{
  "html": {
    "parser": {
      "interpolation": true
    }
  }
}
```

这允许 Biome 正确解析和格式化 Angular 组件模板，而不会将插值语法视为错误。
