Vue-konfiguraatio sisältää Vue.js-kohtaisia linttaussääntöjä komponentin rakenteelle, data-asetuksille, laskennallisille ominaisuuksille ja varatuille avainsanoille. Se sisältää myös `.vue`-tiedostoille tarkoitettuja ohituksia väärien positiivisten linttausvirheiden estämiseksi.

## Asennus

Lisää Vue-konfiguraatio `biome.jsonc`-tiedostoosi:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core", "ultracite/vue"]
}
```

## Yleiskatsaus

Tämä konfiguraatio lisää Vue.js-kehitykseen liittyviä sääntöjä:

- **HTML-tuki**: Ottaa käyttöön kokeellisen täydellisen HTML-tuen, joka sisältää script- ja style-sisennykset
- **React-spesifiset propit**: Estää React-spesifisten propien käytön Vue-komponenteissa
- **.vue-tiedostojen ohitukset**: Lievemmät säännöt `.vue`-tiedostoille Single File Component -rakenteen käsittelyä varten

## HTML-muotoilijan määritykset

Vue-esiasetus ottaa käyttöön kokeellisen HTML-tuen ja tarjoaa erikoismuotoilun upotetuille skripteille ja tyyleille:

```jsonc
{
  "html": {
    "experimentalFullSupportEnabled": true,
    "formatter": {
      "indentScriptAndStyle": true
    }
  }
}
```

Tämä konfiguraatio:
- **Kokeellinen täydellinen tuki**: Mahdollistaa Biomen kokeellisen HTML-parsinta- ja muotoilutoiminnallisuuden `.vue`-tiedostoille
- **Sisennä script ja style**: Sisentää oikein koodin `<script>`- ja `<style>`-tageissa Vue Single File Component -tiedostoissa

## Vue-kohtaiset säännöt

### Epäilyttävät

| Rule | Setting | Description |
|------|---------|-------------|
| `noReactSpecificProps` | `error` | Estä React-spesifisiä proppeja kuten `className` ja `htmlFor` Vuen yhteydessä. Käytä sen sijaan `class` ja `for`. |

## .vue-tiedostojen ohitukset

Seuraavat säännöt on poistettu käytöstä `.vue`-tiedostoissa Biomen osittaisen tuen vuoksi, jotta vältetään väärät positiiviset linttausvirheet Vue Single File Component -tiedostoissa:

### Oikeellisuus

| Rule | Setting | Description |
|------|---------|-------------|
| `noUnusedVariables` | `off` | Sallii käyttämättömien muuttujien esiintymisen `.vue`-tiedostoissa. Skriptiosiossa määritellyt muuttujat saatetaan käyttää vain template-osassa. |
| `noUnusedImports` | `off` | Sallii käyttämättömät importit `.vue`-tiedostoissa. Importit saatetaan käyttää vain template-osassa. |

### Tyyli

| Rule | Setting | Description |
|------|---------|-------------|
| `useConst` | `off` | Sallii `let`- ja `var`-ilmoitukset `.vue`-tiedostoissa reaktiivisten datamallien vuoksi. |
| `useImportType` | `off` | Poistaa eksplisiittisten import-tyyppivaatimusten tarpeen paremman yhteensopivuuden vuoksi Vuen rakennusjärjestelmän kanssa. |
