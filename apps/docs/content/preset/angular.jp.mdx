この Angular の設定は、Angular のテンプレート構文をサポートするための Angular 固有の HTML パーサ設定を含みます。

## インストール

Angular の設定を `biome.jsonc` に追加してください:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core", "ultracite/angular"]
}
```

## 概要

この設定は Angular のテンプレート構文のサポートを追加します:

- **HTML 補間**: Angular テンプレートの補間構文 (`{{ }}`) の解析を有効にします
- **テンプレートディレクティブ**: `*ngIf`、`*ngFor` などの Angular ディレクティブをサポートします
- **プロパティバインディング**: `[property]="value"` のような Angular のプロパティバインディング構文を処理します
- **イベントバインディング**: `(event)="handler()"` のようなイベントバインディング構文をサポートします

## 設定

Angular プリセットは HTML パーサを設定して、Angular のテンプレート補間を認識します:

```jsonc
{
  "html": {
    "parser": {
      "interpolation": true
    }
  }
}
```

これにより、Biome は補間構文をエラーとして扱うことなく、Angular コンポーネントのテンプレートを適切に解析およびフォーマットできます。
