Die Angular-Konfiguration verfügt über Angular-spezifische HTML-Parser-Einstellungen, um die Template-Syntax von Angular zu unterstützen.

## Installation

Fügen Sie die Angular-Konfiguration zu Ihrer `biome.jsonc` hinzu:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core", "ultracite/angular"]
}
```

## Übersicht

Diese Konfiguration fügt Unterstützung für die Template-Syntax von Angular hinzu:

- **HTML-Interpolation**: Ermöglicht das Parsen der Angular-Template-Interpolation-Syntax (`{{ }}`)
- **Template-Direktiven**: Unterstützt Angular-Direktiven wie `*ngIf`, `*ngFor`, usw.
- **Property-Bindung**: Verarbeitet die Angular-Property-Binding-Syntax `[property]="value"`
- **Event-Bindung**: Unterstützt die Event-Bindungssyntax `(event)="handler()"`

## Konfiguration

Das Angular-Preset konfiguriert den HTML-Parser so, dass er die Template-Interpolation von Angular erkennt:

```jsonc
{
  "html": {
    "parser": {
      "interpolation": true
    }
  }
}
```

Dadurch kann Biome Angular-Komponenten-Templates korrekt parsen und formatieren, ohne die Interpolationssyntax als Fehler zu behandeln.
