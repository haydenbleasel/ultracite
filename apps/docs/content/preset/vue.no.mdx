Vue-konfigurasjonen inneholder Vue.js-spesifikke lintregler for komponentstruktur, data-alternativer, beregnede egenskaper og reserverte nøkkelord. Den inkluderer også overstyringer for å forhindre falske positive lintfeil i `.vue`-filer.

## Installasjon

Legg til Vue-konfigurasjonen i din `biome.jsonc`:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core", "ultracite/vue"]
}
```

## Oversikt

Denne konfigurasjonen legger til regler som er spesifikke for Vue.js-utvikling:

- **HTML-støtte**: Aktiverer eksperimentell full HTML-støtte med innrykk for script og style
- **React-spesifikke props**: Forby React-spesifikke props i Vue-komponenter
- **Overstyringer for Vue-filer**: Avslappede regler for `.vue`-filer for å håndtere Single File Component-struktur

## Konfigurasjon for HTML-formattering

Vue-presetet aktiverer eksperimentell HTML-støtte med spesiell formatering for innebygde script- og style-seksjoner:

```jsonc
{
  "html": {
    "experimentalFullSupportEnabled": true,
    "formatter": {
      "indentScriptAndStyle": true
    }
  }
}
```

Denne konfigurasjonen:
- **Eksperimentell full støtte**: Aktiverer Biomes eksperimentelle HTML-parsing og formatteringsmuligheter for `.vue`-filer
- **Innrykk for script og style**: Indenterer riktig kode inne i `<script>`- og `<style>`-tagger i Vue Single File Components

## Vue-spesifikke regler

### Mistenkelig

| Rule | Setting | Description |
|------|---------|-------------|
| `noReactSpecificProps` | `error` | Forby React-spesifikke props som `className` og `htmlFor` i Vue. Bruk `class` og `for` i stedet. |

## Overstyringer for Vue-filer

Følgende regler er deaktivert for `.vue`-filer for å forhindre falske positive på grunn av Biomes delvise støtte for Vue Single File Components:

### Korrekthet

| Rule | Setting | Description |
|------|---------|-------------|
| `noUnusedVariables` | `off` | Tillat ubrukte variabler i `.vue`-filer. Variabler i script-delen kan kun brukes i malen. |
| `noUnusedImports` | `off` | Tillat ubrukte importer i `.vue`-filer. Importer kan kun brukes i malseksjonen. |

### Stil

| Rule | Setting | Description |
|------|---------|-------------|
| `useConst` | `off` | Tillat `let` og `var`-deklarasjoner i `.vue`-filer for reaktive datamønstre. |
| `useImportType` | `off` | Deaktiver krav om eksplisitt importtype for bedre kompatibilitet med Vues byggesystem. |
