Vue yapılandırması, bileşen yapısı, data seçenekleri, computed özellikleri ve ayrılmış anahtar kelimeler için Vue.js'e özgü lint kuralları içerir. Ayrıca `.vue` dosyalarında yanlış pozitif lint hatalarını önlemek için geçersiz kılmalar (overrides) içerir.

## Kurulum

Vue yapılandırmasını `biome.jsonc` dosyanıza ekleyin:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core", "ultracite/vue"]
}
```

## Genel Bakış

Bu yapılandırma Vue.js geliştirmesine özgü kurallar ekler:

- **HTML Desteği**: Script ve style girintilemesi ile deneysel tam HTML desteğini etkinleştirir
- **React'e Özgü Props**: Vue bileşenlerinde React'e özgü prop'ları yasaklar
- **Vue Dosyası Geçersiz Kılmaları**: Tek Dosya Bileşen yapısını ele almak için `.vue` dosyalarında gevşetilmiş kurallar

## HTML Biçimlendirici Yapılandırması

Vue ön ayarı, gömülü scriptler ve stiller için özel biçimlendirme ile deneysel HTML desteğini etkinleştirir:

```jsonc
{
  "html": {
    "experimentalFullSupportEnabled": true,
    "formatter": {
      "indentScriptAndStyle": true
    }
  }
}
```

Bu yapılandırma:
- **Deneysel Tam Destek**: `.vue` dosyaları için Biome'un deneysel HTML ayrıştırma ve biçimlendirme yeteneklerini etkinleştirir
- **Script ve Style İçin Girintileme**: Vue Tek Dosya Bileşenlerindeki `<script>` ve `<style>` etiketleri içindeki kodu doğru şekilde girintiler

## Vue'ye Özgü Kurallar

### Şüpheli

| Kural | Ayar | Açıklama |
|------|---------|-------------|
| `noReactSpecificProps` | `error` | Vue'de `className` ve `htmlFor` gibi React'e özgü prop'ları yasaklar. Bunun yerine `class` ve `for` kullanın. |

## Vue Dosyası Geçersiz Kılmaları

Aşağıdaki kurallar, Biome'un Vue Tek Dosya Bileşenleri için kısmi desteğinden kaynaklanan yanlış pozitifleri önlemek amacıyla `.vue` dosyaları için devre dışı bırakılmıştır:

### Doğruluk

| Kural | Ayar | Açıklama |
|------|---------|-------------|
| `noUnusedVariables` | `off` | `.vue` dosyalarında kullanılmayan değişkenlere izin verir. Script bölümündeki değişkenler yalnızca template içinde kullanılabilir. |
| `noUnusedImports` | `off` | `.vue` dosyalarında kullanılmayan importlara izin verir. Importlar yalnızca template bölümünde kullanılabilir. |

### Stil

| Kural | Ayar | Açıklama |
|------|---------|-------------|
| `useConst` | `off` | Reaktif veri desenleri için `.vue` dosyalarında `let` ve `var` bildirimlerine izin verir. |
| `useImportType` | `off` | Vue'nun derleme sistemiyle daha iyi uyumluluk için açık import tipi gereksinimlerini devre dışı bırakır. |
