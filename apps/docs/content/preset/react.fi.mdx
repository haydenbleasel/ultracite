React-konfiguraatio sisältää React-spesifisiä linttaussääntöjä JSX:lle, hookeille ja komponenttimalleille.

## Asennus

Lisää React-konfiguraatio `biome.jsonc`-tiedostoosi:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core", "ultracite/react"]
}
```

## Yleiskatsaus

Tämä konfiguraatio lisää React-kehitykseen liittyviä sääntöjä:

- **JSX-säännöt**: Pakottaa oikean JSX-syntaksin ja -käytännöt
- **Hook-säännöt**: Varmistaa, että React-hookeja käytetään oikein
- **Komponenttisäännöt**: Soveltaa React-komponenttien parhaita käytäntöjä
- **Children-käsittely**: Estää virheellisen children-propin käytön

## React-spesifiset säännöt

### Oikeellisuus

| Sääntö | Asetus | Kuvaus |
|------|---------|-------------|
| `noChildrenProp` | `error` | Estä children-propin välittäminen propsina. children tulisi olla avaus- ja sulkutagien välissä. |
| `noNestedComponentDefinitions` | `error` | Estä komponenttimäärittelyt muiden komponenttien sisällä. Tämä aiheuttaa tarpeettomia uudelleenrenderöintejä. |
| `noReactPropAssignments` | `error` | Estä props-arvojen uudelleenmäärittely React-komponenteissa. Propsia tulisi käsitellä muuttumattomina. |
| `noRenderReturnValue` | `error` | Estä `ReactDOM.render()`-funktion paluuarvon käyttäminen. |
| `noVoidElementsWithChildren` | `error` | Estä, että tyhjillä elementeillä (esim. `<img>`, `<br>`) olisi children. |
| `useExhaustiveDependencies` | `error` | Pakota, että kaikki riippuvuudet on määritelty oikein React-hookeissa (useEffect, useCallback, useMemo). |
| `useHookAtTopLevel` | `error` | Pakota, että kaikki React-hookit kutsutaan komponenttifunktioiden ylimmällä tasolla, ei silmukoiden, ehtojen tai sisäkkäisten funktioiden sisällä. |
| `useJsxKeyInIterable` | `error` | Varmista, että iteroitavilla elementeillä on `key`-prop Reactin reconciliota varten. |

### Tyyli

| Sääntö | Asetus | Kuvaus |
|------|---------|-------------|
| `useReactFunctionComponents` | `error` | Edellytä funktionaalisten komponenttien käyttöä luokkakomponenttien sijaan Reactissa. |

### Epäilyttävät

| Sääntö | Asetus | Kuvaus |
|------|---------|-------------|
| `noArrayIndexKey` | `error` | Estä taulukon indeksien käyttöä key-arvoina. Indeksit eivät ole pysyviä tunnisteita ja voivat aiheuttaa ongelmia komponentin tilassa. |
| `noDuplicateJsxProps` | `error` | Estä samojen ominaisuuksien toistuminen JSX:ssä. |
| `noSuspiciousSemicolonInJsx` | `error` | Estä puolipisteet, jotka muuttavat JSX-elementtien semantiikkaa. |
| `noReactSpecificProps` | `off` | Salli React-spesifiset propit kuten `className` ja `htmlFor`. |
