Vue-konfigurationen innehåller lintningsregler som är specifika för Vue.js gällande komponentstruktur, dataalternativ, beräknade egenskaper och reserverade nyckelord. Den inkluderar även överstyrningar för att förhindra falska positiva lintningsfel i `.vue`-filer.

## Installation

Lägg till Vue-konfigurationen i din `biome.jsonc`:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core", "ultracite/vue"]
}
```

## Översikt

Denna konfiguration lägger till regler som är specifika för Vue.js-utveckling:

- **HTML-stöd**: Aktiverar experimentellt fullständigt HTML-stöd med indrag för script och style
- **React-specifika props**: Förbjud React-specifika props i Vue-komponenter
- **Överstyrningar för Vue-filer**: Lättade regler för `.vue`-filer för att hantera Single File Component-strukturen

## Konfiguration för HTML-formattering

Vue-preseten aktiverar experimentellt HTML-stöd med speciell formatering för inbäddade skript och stilar:

```jsonc
{
  "html": {
    "experimentalFullSupportEnabled": true,
    "formatter": {
      "indentScriptAndStyle": true
    }
  }
}
```

Denna konfiguration:
- **Experimentellt fullstöd**: Aktiverar Biome:s experimentella HTML-parsnings- och formateringsfunktioner för `.vue`-filer
- **Indrag för script och style**: Indenterar korrekt kod inuti `<script>`- och `<style>`-taggar i Vue Single File Components

## Vue-specifika regler

### Misstänkta

| Regel | Inställning | Beskrivning |
|------|---------|-------------|
| `noReactSpecificProps` | `error` | Förbjud React-specifika props som `className` och `htmlFor` i Vue. Använd istället `class` och `for`. |

## Överstyrningar för .vue-filer

Följande regler är inaktiverade för `.vue`-filer för att förhindra falska positiva resultat på grund av Biome:s partiella stöd för Vue Single File Components:

### Korrekthet

| Regel | Inställning | Beskrivning |
|------|---------|-------------|
| `noUnusedVariables` | `off` | Tillåt oanvända variabler i `.vue`-filer. Variabler i scriptsektionen kan endast användas i templaten. |
| `noUnusedImports` | `off` | Tillåt oanvända imports i `.vue`-filer. Importer kan endast användas i template-sektionen. |

### Stil

| Regel | Inställning | Beskrivning |
|------|---------|-------------|
| `useConst` | `off` | Tillåt `let`- och `var`-deklarationer i `.vue`-filer för reaktiva datamönster. |
| `useImportType` | `off` | Inaktivera krav på explicit importtyp för bättre kompatibilitet med Vues byggsystem. |
