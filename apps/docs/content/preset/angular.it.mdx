La configurazione per Angular include impostazioni specifiche del parser HTML per supportare la sintassi dei template di Angular.

## Installazione

Aggiungi la configurazione Angular al tuo `biome.jsonc`:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core", "ultracite/angular"]
}
```

## Panoramica

Questa configurazione aggiunge il supporto per la sintassi dei template di Angular:

- **Interpolazione HTML**: Abilita il parsing della sintassi di interpolazione dei template di Angular (`{{ }}`)
- **Direttive dei template**: Supporta direttive Angular come `*ngIf`, `*ngFor`, ecc.
- **Binding delle proprietà**: Gestisce la sintassi di binding delle proprietà Angular `[property]="value"`
- **Binding degli eventi**: Supporta la sintassi di binding degli eventi `(event)="handler()"`

## Configurazione

Il preset Angular configura il parser HTML per riconoscere l'interpolazione dei template di Angular:

```jsonc
{
  "html": {
    "parser": {
      "interpolation": true
    }
  }
}
```

Questo permette a Biome di analizzare e formattare correttamente i template dei componenti Angular senza considerare la sintassi di interpolazione come errori.
