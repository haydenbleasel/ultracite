Vue konfiguracija sadrži pravila za linting specifična za Vue.js za strukturu komponenti, opcije podataka, izračunata svojstva i rezervisane ključne reči. Takođe uključuje override-e kako bi se sprečile lažno pozitivne greške lintovanja u `.vue` fajlovima.

## Instalacija

Dodajte Vue konfiguraciju u vaš `biome.jsonc`:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core", "ultracite/vue"]
}
```

## Pregled

Ova konfiguracija dodaje pravila specifična za Vue.js razvoj:

- **Podrška za HTML**: Omogućava eksperimentalnu potpunu podršku za HTML sa uvlačenjem sadržaja unutar `<script>` i `<style>` tagova
- **React-specifični props**: Zabranjuje React-specifične props u Vue komponentama
- **Override za Vue fajlove**: Opuštenija pravila za `.vue` fajlove kako bi se obradila struktura Single File Component-a

## Konfiguracija HTML formatiranja

Vue preset omogućava eksperimentalnu podršku za HTML sa specijalnim formatiranjem za ugrađene skripte i stilove:

```jsonc
{
  "html": {
    "experimentalFullSupportEnabled": true,
    "formatter": {
      "indentScriptAndStyle": true
    }
  }
}
```

Ova konfiguracija:
- **Eksperimentalna potpuna podrška**: Omogućava Biome-ove eksperimentalne mogućnosti parsiranja i formatiranja HTML-a za `.vue` fajlove
- **Uvlačenje skripti i stilova**: Ispravno uvlači kod unutar `<script>` i `<style>` tagova u Vue Single File Component-ima

## Vue-specifična pravila

### Sumnjivo

| Pravilo | Podešavanje | Opis |
|---------|------------|------|
| `noReactSpecificProps` | `error` | Zabranjuje React-specifične props kao što su `className` i `htmlFor` u Vue-u. Umesto njih koristite `class` i `for`. |

## Override za Vue fajlove

Sledeća pravila su onemogućena za `.vue` fajlove kako bi se sprečile lažno pozitivne greške zbog Biome-ove delimične podrške za Vue Single File Components:

### Ispravnost

| Pravilo | Podešavanje | Opis |
|---------|------------|------|
| `noUnusedVariables` | `off` | Dozvoljava neiskorišćene promenljive u `.vue` fajlovima. Promenljive u <script> sekciji mogu se koristiti samo u `<template>` delu. |
| `noUnusedImports` | `off` | Dozvoljava neiskorišćene import-e u `.vue` fajlovima. Importi mogu se koristiti samo u `<template>` delu. |

### Stil

| Pravilo | Podešavanje | Opis |
|---------|------------|------|
| `useConst` | `off` | Dozvoljava `let` i `var` deklaracije u `.vue` fajlovima za obrasce reaktivnih podataka. |
| `useImportType` | `off` | Onemogućava zahteve za eksplicitnim tipovima import-a radi bolje kompatibilnosti sa Vue build sistemom. |
