Angular-konfigurationen innehåller Angular-specifika inställningar för HTML-parsern för att stödja Angulars templatesyntax.

## Installation

Lägg till Angular-konfigurationen i din `biome.jsonc`:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core", "ultracite/angular"]
}
```

## Översikt

Denna konfiguration lägger till stöd för Angulars templatesyntax:

- **HTML-interpolering**: Aktiverar parsning av Angulars template-interpoleringssyntax (`{{ }}`)
- **Templatedirektiv**: Stöder Angular-direktiv som `*ngIf`, `*ngFor`, osv.
- **Egenskapsbindning**: Hanterar Angulars syntax för egenskapsbindning `[property]="value"`
- **Händelsebindning**: Stöder syntax för händelsebindning `(event)="handler()"`

## Konfiguration

Angular-preseten konfigurerar HTML-parsern för att känna igen Angulars template-interpolering:

```jsonc
{
  "html": {
    "parser": {
      "interpolation": true
    }
  }
}
```

Detta gör att Biome korrekt kan parsa och formatera Angular-komponentmallar utan att tolka interpoleringssyntaxen som fel.
