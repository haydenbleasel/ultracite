title: Svelte

---

Tá an chumraíocht Svelte sonrach don Svelte. Is réamhshocruithe í seo ar réamhíosta toisc nach bhfuil rialacha lintithe ar leith do Svelte fós i Biome, ach tá osclaíochtaí tábhachtacha inti chun earráidí dearfacha bréagacha lintithe i gcomhaid `.svelte` a chosc.

## Suiteáil

Cuir an chumraíocht Svelte leis i do `biome.jsonc`:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core", "ultracite/svelte"]
}
```

## Forbhreathnú

Tá an chumraíocht seo:

- **Tacaíocht HTML**: Cumasaíonn sí tacaíocht iomlán HTML turgnamhach le cuimsiú do <script> agus <style>
- **Airíonna Shonracha React**: Marcálann sí airíonna shonracha React cosúil le `className` agus `htmlFor` ós rud é go n-úsáideann Svelte tréithe HTML caighdeánacha (`class` agus `for`)
- **Socrúcháin do Chomhaid `.svelte`**: Rialacha faoisimhe do chomhaid `.svelte` chun struchtúr chomhpháirte Svelte a láimhseáil

## Cumraíocht Formáidithe HTML

Cuirtear tacaíocht HTML turgnamhach leis an réamhshocrú Svelte le formáidiú speisialta do scriptí agus stíleanna comhtháthaithe:

```jsonc
{
  "html": {
    "experimentalFullSupportEnabled": true,
    "formatter": {
      "indentScriptAndStyle": true
    }
  }
}
```

Tá an chumraíocht seo:
- **Tacaíocht Iomlán Turgnamhach**: Cumasaíonn sé cumais pharsála agus formáidithe HTML turgnamhacha Biome do chomhaid `.svelte`
- **Cuimsigh Script agus Style**: Déanann sé cód laistigh de thaighde `<script>` agus `<style>` i gcomhpháirteanna Svelte a chuisniú i gceart

## Rialacha Sonracha Svelte

### Amhrasach

| Rialacha | Socrú | Cur síos |
|------|---------|-------------|
| `noReactSpecificProps` | `error` | Cosc ar airíonna shonracha React cosúil le `className` agus `htmlFor` i Svelte. Úsáid `class` agus `for` ina áit. |

## Socrúcháin Comhaid Svelte

Tá na rialacha seo a leanas díghníomhachtaithe do chomhaid `.svelte` chun dearbhaintí dearfacha a chosc mar gheall ar shuíomh páirteach Biome maidir le comhpháirteanna Svelte:

### Ceartas

| Rialacha | Socrú | Cur síos |
|------|---------|-------------|
| `noUnusedVariables` | `off` | Ceadaigh athróga nach n-úsáidtear i gcomhaid `.svelte`. Is féidir go n-úsáidfear athróga sa chuid script ach amháin sa teimpléad. |
| `noUnusedImports` | `off` | Ceadaigh iompórtálacha nach n-úsáidtear i gcomhaid `.svelte`. D’fhéadfadh iompórtálacha a bheith in úsáid ach amháin sa chuid teimpléid. |

### Stíl

| Rialacha | Socrú | Cur síos |
|------|---------|-------------|
| `useConst` | `off` | Ceadaigh fhoráilithe `let` agus `var` i gcomhaid `.svelte` do ráitis imoibríocha. |
| `useImportType` | `off` | Díghníomhachtaigh ceanglais chineál iompórtála shoiléire le haghaidh comhoiriúnacht níos fearr le córas tógála Svelte. |
