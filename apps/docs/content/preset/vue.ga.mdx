Tá sa chumraíocht Vue rialacha iniúchta shonracha do Vue.js le haghaidh struchtúir comhpháirte, roghanna sonraí, airíonna ríofa, agus eochairfhocail á gcoimeád i leataobh. Tá athshocruithe ann freisin chun earráidí iniúchta dearfacha bréagacha a chosc i gcomhaid `.vue`.

## Suiteáil

Cuir an chumraíocht Vue leis do `biome.jsonc`:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core", "ultracite/vue"]
}
```

## Forbhreathnú

Cuireann an chumraíocht seo rialacha sonracha do fhorbairt Vue.js leis:

- **Tacaíocht HTML**: Cuireann sí tacaíocht iomlán thriailteach don HTML ar chumas le h-indéantáil do script agus style
- **Airíonna Sonracha React**: Ní cheadaítear airíonna sonracha React i gcomhpháirteanna Vue
- **Athshocruithe do chomhaid Vue**: Rialacha faoisithe do chomhaid `.vue` chun struchtúr Single File Component a láimhseáil

## Cumraíocht Formáidóra HTML

Cuireann réamhshocraithe Vue tacaíocht HTML thriailteach ar chumas le formáidiú speisialta do scriptí agus stíleanna atá ionsuite:

```jsonc
{
  "html": {
    "experimentalFullSupportEnabled": true,
    "formatter": {
      "indentScriptAndStyle": true
    }
  }
}
```

Sa chumraíocht seo:
- **Tacaíocht Iomlán Trialaithe**: Cumasaíonn sé cumais anailíse agus formáide HTML thriailte Biome do chomhaid `.vue`
- **Indeantáil Script agus Style**: Déanann sé innéantáil cheart ar an gcód laistigh de thaganna `<script>` agus `<style>` i Single File Components Vue

## Rialacha Sonracha Vue

### Amhrasach

| Riail | Socrú | Cur síos |
|------|---------|-------------|
| `noReactSpecificProps` | `error` | Ní cheadaítear airíonna sonracha React cosúil le `className` agus `htmlFor` i Vue. Úsáid `class` agus `for` ina áit. |

## Athshocruithe do chomhaid Vue

Tá na rialacha seo a leanas díghníomhachtaithe do chomhaid `.vue` chun dearbhuithe bhréagacha a chosc de bharr tacaíocht pháirteach Biome do Single File Components Vue:

### Ceartas

| Riail | Socrú | Cur síos |
|------|---------|-------------|
| `noUnusedVariables` | `off` | Ceadaíonn sé athróganna nach n-úsáidtear i gcomhaid `.vue`. D’fhéadfadh go n-úsáidfear athróganna sa rannán script ach amháin sa teimpléad. |
| `noUnusedImports` | `off` | Ceadaíonn sé allmhairí nach n-úsáidtear i gcomhaid `.vue`. D’fhéadfadh allmhairí a úsáidtear ach sa rannán teimpléid. |

### Stíl

| Riail | Socrú | Cur síos |
|------|---------|-------------|
| `useConst` | `off` | Ceadaíonn sé fógraí `let` agus `var` i gcomhaid `.vue` do phatrúin sonraí imoibríocha. |
| `useImportType` | `off` | Múchann sé riachtanais chineála allmhairiúcháin shonracha chun comhoiriúnacht níos fearr le córas tógála Vue a bhaint amach. |
