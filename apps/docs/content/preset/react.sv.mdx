React-konfigurationen innehåller React-specifika lintregler för JSX, hooks och komponentmönster.

## Installation

Lägg till React-konfigurationen i din `biome.jsonc`:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core", "ultracite/react"]
}
```

## Översikt

Denna konfiguration lägger till regler specifika för React-utveckling:

- **JSX-regler**: Upprätthåll korrekt JSX-syntax och mönster
- **Hooks-regler**: Säkerställ att React-hooks används korrekt
- **Komponentregler**: Upprätthåll bästa praxis för React-komponenter
- **Hantering av children**: Förhindra felaktig användning av children-prop

## React-specifika regler

### Korrekthet

| Regel | Inställning | Beskrivning |
|------|---------|-------------|
| `noChildrenProp` | `error` | Förhindra att children passeras som props. children bör nästas mellan öppnings- och stängningstaggarna. |
| `noNestedComponentDefinitions` | `error` | Förhindra komponentdefinitioner inuti andra komponenter. Detta orsakar onödiga omrenderingar. |
| `noReactPropAssignments` | `error` | Förhindra att props tilldelas om i React-komponenter. Props bör behandlas som immutabla. |
| `noRenderReturnValue` | `error` | Förhindra användning av returvärdet från `ReactDOM.render()`. |
| `noVoidElementsWithChildren` | `error` | Förhindra att void-element (som `<img>`, `<br>`) har children. |
| `useExhaustiveDependencies` | `error` | Säkerställ att alla beroenden är korrekt angivna i React-hooks (`useEffect`, `useCallback`, `useMemo`). |
| `useHookAtTopLevel` | `error` | Säkerställ att alla React-hooks anropas från topplinjen i komponentfunktioner, inte inne i loopar, villkor eller nästlade funktioner. |
| `useJsxKeyInIterable` | `error` | Säkerställ att element i itererbara strukturer har en `key`-prop för Reacts rekonsiliering. |

### Stil

| Regel | Inställning | Beskrivning |
|------|---------|-------------|
| `useReactFunctionComponents` | `error` | Tvinga användning av funktionskomponenter istället för klasskomponenter i React. |

### Misstänkta

| Regel | Inställning | Beskrivning |
|------|---------|-------------|
| `noArrayIndexKey` | `error` | Förhindra användning av arrayindex som keys. Arrayindex är inte stabila identifierare och kan orsaka problem med komponenternas tillstånd. |
| `noDuplicateJsxProps` | `error` | Förhindra duplicerade egenskaper i JSX. |
| `noSuspiciousSemicolonInJsx` | `error` | Förhindra semikolon som ändrar semantiken i JSX-element. |
| `noReactSpecificProps` | `off` | Tillåt React-specifika props som `className` och `htmlFor`. |
