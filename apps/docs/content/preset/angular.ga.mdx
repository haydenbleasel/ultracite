---

Tá socruithe parser HTML shonracha do Angular sa chumraíocht Angular chun tacú le sintéis teimpléid Angular.

## Suiteáil

Cuir an chumraíocht Angular leis do `biome.jsonc`:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core", "ultracite/angular"]
}
```

## Forbhreathnú

Cuiríonn an chumraíocht seo tacaíocht le sintéis teimpléid Angular:

- **Idirmheánú HTML**: Cumasaíonn sé an sintéis idirmheánaithe teimpléid Angular (`{{ }}`) a pharsáil
- **Treoracha Teimpléid**: Tacaíonn sé le treoracha Angular cosúil le `*ngIf`, `*ngFor`, srl.
- **Ceangail Airíonna**: Bainistíonn sé sintéis cheangail airíonna Angular `[property]="value"`
- **Ceangail Imeachtaí**: Tacaíonn sé le sintéis cheangail imeachtaí `(event)="handler()"`

## Cumraíocht

Socraíonn an réamhshocrú Angular an parser HTML chun idirmheánú teimpléid Angular a aithint:

```jsonc
{
  "html": {
    "parser": {
      "interpolation": true
    }
  }
}
```

Ligeann sé seo do Biome teimpléid chomhpháirte Angular a pharsáil agus a fhormáidiú i gceart gan an sintéis idirmheánaithe a chóireáil mar earráidí.
