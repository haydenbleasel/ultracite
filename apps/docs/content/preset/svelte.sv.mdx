---
title: Svelte
description: Svelte-specifik konfiguration för Ultracite.
---

Svelte-konfigurationen innehåller Svelte-specifika inställningar. Detta är ett minimalt preset eftersom Biome ännu inte har dedikerade Svelte-specifika lintningsregler, men det inkluderar viktiga överstyrningar för att förhindra falska positiva lintfel i `.svelte`-filer.

## Installation

Lägg till Svelte-konfigurationen i din `biome.jsonc`:

```jsonc title="biome.jsonc"
{
  "extends": ["ultracite/core", "ultracite/svelte"]
}
```

## Översikt

Denna konfiguration:

- **HTML-stöd**: Aktiverar experimentellt fullständigt HTML-stöd med indentering i script- och style-taggar
- **React-specifika props**: Markerar React-specifika props som `className` och `htmlFor` eftersom Svelte använder standard HTML-attribut (`class` och `for`)
- **Överstyrningar för Svelte-filer**: Mjukare regler för `.svelte`-filer för att hantera Sveltes komponentstruktur

## Konfiguration för HTML-formatteraren

Svelte-preseten aktiverar experimentellt HTML-stöd med särskild formatering för inbäddade script- och style-block:

```jsonc
{
  "html": {
    "experimentalFullSupportEnabled": true,
    "formatter": {
      "indentScriptAndStyle": true
    }
  }
}
```

Denna konfiguration:
- **Experimentellt fullständigt stöd**: Aktiverar Biome:s experimentella HTML-parsnings- och formatteringsfunktioner för `.svelte`-filer
- **Indentering av script och style**: Indenterar korrekt kod inne i `<script>`- och `<style>`-taggar i Svelte-komponenter

## Svelte-specifika regler

### Misstänkta

| Regel | Inställning | Beskrivning |
|------|---------|-------------|
| `noReactSpecificProps` | `error` | Förbjud React-specifika props som `className` och `htmlFor` i Svelte. Använd `class` och `for` istället. |

## Överstyrningar för Svelte-filer

Följande regler är inaktiverade för `.svelte`-filer för att förhindra falska positiva felmeddelanden på grund av Biome:s partiella stöd för Svelte-komponenter:

### Korrekthet

| Regel | Inställning | Beskrivning |
|------|---------|-------------|
| `noUnusedVariables` | `off` | Tillåt oanvända variabler i `.svelte`-filer. Variabler i scriptsektionen kan endast användas i mallsektionen. |
| `noUnusedImports` | `off` | Tillåt oanvända importer i `.svelte`-filer. Importer kan endast användas i mallsektionen. |

### Stil

| Regel | Inställning | Beskrivning |
|------|---------|-------------|
| `useConst` | `off` | Tillåt `let` och `var`-deklarationer i `.svelte`-filer för reaktiva satser. |
| `useImportType` | `off` | Inaktivera krav på explicit importtyp för bättre kompatibilitet med Sveltes byggsystem. |
